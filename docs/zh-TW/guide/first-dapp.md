# 第一次使用 DApp

## 什麼是 DApp？

### 一句話解釋

```
DApp = Decentralized Application = 去中心化應用

與傳統 App 的區別：

傳統 App：
用戶 → 公司伺服器 → 公司資料庫
         ↓
    公司可以刪除你的帳號

DApp：
用戶 → 區塊鏈（智能合約）
         ↓
    沒有人能刪除你的帳號
```

## 使用 DApp 的準備工作

### 1. 安裝錢包

```
推薦錢包：
- MetaMask（瀏覽器插件）
- Trust Wallet（手機）

以 MetaMask 為例：
1. 訪問 metamask.io
2. 點擊"下載"
3. 選擇瀏覽器擴展
4. 按提示創建錢包
5. 備份助記詞！
```

### 2. 獲取代幣

```
DApp 使用需要 Gas 費

BSC 網路：
- 需要 BNB 支付 Gas
- 可以從交易所提現 BNB 到錢包

獲取步驟：
1. 交易所買入 BNB
2. 提現到錢包地址
3. 選擇 BSC (BEP20) 網路
```

## 連接 DApp

### MetaMask 連接示例

```
步驟：
1. 訪問 DApp 網站
   例如：pancakeswap.finance

2. 點擊"Connect Wallet"
   通常在右上角

3. 選擇錢包
   選擇 MetaMask

4. 確認連接
   在 MetaMask 彈窗點擊"確認"

5. 連接成功
   錢包地址顯示在頁面上
```

### 常見問題

```
Q：錢包顯示"連接失敗"
A：檢查錢包是否解鎖，網路是否正確

Q：找不到錢包選項
A：確保錢包擴展已啟用

Q：一直彈出確認
A：錢包可能有多筆待處理交易，先取消
```

## 使用 DApp 的基本操作

### 1. 質押/存入

```
假設：使用 Staking DApp

1. 連接錢包

2. 選擇質押數量
   - 點擊輸入框
   - 輸入數量或點擊"MAX"

3. 點擊"Stake"按鈕

4. MetaMask 彈窗確認
   - 查看 Gas 費用
   - 點擊"確認"

5. 等待區塊確認
   通常 3-30 秒
```

### 2. 交易/交換

```
假設：使用 DEX 交換代幣

1. 連接錢包

2. 選擇交換對
   例如：BNB → USDT

3. 輸入數量
   查看預計收到數量

4. 點擊"Swap"

5. 確認交易
   - 檢查滑點設置
   - 確認 Gas 費用

6. 完成交換
```

### 3. 領取收益

```
假設：領取 Staking 收益

1. 進入 Staking 頁面

2. 查看收益
   - 顯示待領取數量
   - APY 年化收益率

3. 點擊"Claim"或"Harvest"

4. 確認交易
   收益將轉入錢包
```

## 交易設定詳解

### Gas 費用

```
Gas = 區塊鏈處理費用

Gas 費用 = Gas 單價 × Gas 消耗

Gas 單價（以 BSC 為例）：
- 緩慢：5 Gwei，便宜但慢
- 一般：10 Gwei，平衡
- 快速：20+ Gwei，貴但快

選擇建議：
- 小額不急：選緩慢，省錢
- 大額或緊急：選快速
```

### 滑點容差

```
滑點 = 預期價格與實際成交價的差異

例如：
- 你期望以 100 USDT 買入
- 實際以 99.5 USDT 成交
- 滑點 = 0.5%

設置建議：
- 穩定幣交易：0.1-0.5%
- 普通代幣：0.5-1%
- 新興代幣：1-3%
```

### 交易截止時間

```
交易只在區塊內有效

如果區塊時間內未能確認
交易自動取消

適用場景：
- 網路擁堵時
- 價格波動大時
```

## 安全檢查清單

### 操作前檢查

```
✅ 確認網址正確
   - 使用書籤或正確輸入
   - 小心釣魚網站

✅ 確認網路正確
   - BSC ≠ ETH
   - 錯網路可能丟幣

✅ 確認合約地址
   - 從官方渠道獲取
   - 不要點擊陌生連結

✅ 小額測試
   - 先用小額測試流程
   - 確認沒問題再加大額
```

### 授權風險

```
有些 DApp 需要"授權"

授權 = 允許合約動用你的代幣

風險：
- 被盜風險
- 授權額度過大

防範：
- 只授權需要的數量
- 使用 Revoke.cash 取消不需要的授權
- 定期檢查授權列表
```

## PulsePay 使用示例

### 體驗完整的 DeFi 流程

```
1. 連接錢包
   訪問 app.pulsepay.fun
   點擊"連接錢包"

2. 充值 USDT
   從交易所提現 USDT 到錢包
   選擇 BSC 網路

3. 兌換 AIP
   訪問 stake.pulsepay.fun
   使用 USDT 兌換 AIP

4. 啟動 AIP 收益分成
   啟動 AIP 收益分成獲得收益
   每日領取分紅

5. 領取收益
   點擊 Claim 領取 USDT/BNB 分紅
```
