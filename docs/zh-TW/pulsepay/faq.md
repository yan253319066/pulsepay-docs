# 常見問題 (FAQ)

## 關於 PulsePay

### PulsePay 是什麼？

**PulsePay Protocol** 是首個將 AI 使用行為轉化為鏈上可分配收益的基礎設施協議。通過整合 AI Gateway、加密支付系統與數字錢包，構建了一套完整的"AI 收費 → 收入沉澱 → Token 分配"閉環經濟體系。

### PulsePay 合法嗎？

PulsePay 是一個技術協議，核心功能是：
- 提供 AI 調用服務
- 提供支付服務
- 提供收益分成服務

我們遵守運營地區的法律法規，並採取合規措施。

### PulsePay 團隊是誰？

團隊資訊將在 Phase 3 階段逐步公開。目前採取以下信任建立措施：
- 產品已上線可驗證
- 合約代碼開源
- 收入鏈上透明
- 漸進式去中心化

## 關於 AIP Token

### AIP 是什麼？

AIP (AI Pulse) 是 PulsePay 協議的治理代幣，持有者可以：
- 參與收益分成
- 分享平台業務增長紅利
- 未來參與協議治理

### AIP 總供應量是多少？

100,000,000 AIP（1 億枚），固定總量，無增發機制。

### AIP 的價值來源是什麼？

AIP 有多重價值支撐：
1. **使用價值** - 質押參與收益分成
2. **通縮機制** - 30% 收入回購銷毀
3. **需求驅動** - 用戶需要 AIP 參與生態

### Phase 1 沒來得及怎麼辦？

Phase 2 開放後可以在 DEX（如 PancakeSwap）購買 AIP，屆時價格將由市場供需決定。

## 關於收益

### 收益從哪裡來？

收益來自 PulsePay 平台的真實業務收入：
- AI Gateway 調用手續費
- 支付系統處理手續費

不是龐氏結構，不是通脹挖礦。

### 收益是多少？

收益取決於：
- 每日平台業務收入
- 你的 AIP 質押量
- 全網 AIP 質押總量

計算公式：
```
個人收益 = (個人質押量 / 全網質押量) × (當日平台收入 × 40%)
```

### 收益怎麼領取？

需要手動點擊"領取"按鈕，收益會轉入你的錢包。

### 收益需要納稅嗎？

取決於你所在國家/地區的稅務法規。建議諮詢當地稅務顧問。

## 關於質押

### 最低質押多少？

最低 10 AIP 起。

### 有鎖定期嗎？

沒有鎖定期，可以隨時質押、隨時解除。

### 解除質押會影響已領取的收益嗎？

不影響。已領取的收益歸你所有，解除質押只影響未來的新收益。

### 質押有風險嗎？

主要風險：
- AIP 價格波動
- 平台業務下降

不存在：
- 爆倉風險
- 清算風險
- 懲罰機制
