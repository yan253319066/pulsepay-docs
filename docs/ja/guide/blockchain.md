# ブロックチェーンとは？

## 一つのストーリーで理解する

### 従来の记账 vs ブロックチェーン记账

友達と共通の账簿をつける必要があるとします：

**従来方式（集中型）：**
```
账簿係りに小明を選んだ
全ての账簿は小明のノートに记录
問題：万一小明が不正したら？
```

**ブロックチェーン方式（分散型）：**
```
全員が账簿を持つ
每次の取引は全員にbroadcast
全員同時に记账、账簿が一致
問題：谁が正确に记忆？说論来吧！
```

## ブロックチェーンのコア原則

### 1. 🧱 ブロックとは？

Excelの行のように、各ブロックには：

```
┌─────────────────────────────┐
│         ブロック #12345     │
├─────────────────────────────┤
│ タイムスタンプ: 2026-03-30 │
├─────────────────────────────┤
│ 取引1: 王さん → 李さん 1 BTC│
│ 取引2: 張さん → 王さん 0.5 │
│ 取引3: ...                 │
├─────────────────────────────┤
│ 前ブロックハッシュ: abc123..│
│ このブロックハッシュ: def456│
└─────────────────────────────┘
```

### 2. 🔗 チェーンの形成

```
ブロック12345 → ブロック12346 → ブロック12347 → ...
     ↓            ↓            ↓
   ハッシュ参照  ハッシュ参照  ハッシュ参照
```

各ブロックは「前のブロックの指紋（ハッシュ）」を含む、DNAのように！

## ブロックチェーンの種類

| 種類 | 代表例 | 特徴 | 速度 |
|------|--------|------|------|
| **パブリックチェーン** | ビットコイン、イーサリアム | 誰でも参加可能 | 遅い |
| **コンソーシアムチェーン** | BSC、Polygon | 许可制ノード参加 | 速い |
| **プライベートチェーン** | 企業内部 | 単一管理 | 最速 |

### BSC（币安スマートチェーン）とは？

```
BSC = Binance Smart Chain

例え：
- イーサリアム = 有料高速道路
- BSC = 幣安が建設した別の高速道路
  - 通行料が安い
  - 速度が速い
  - イーサリアム互換

PulsePayはBSC上で運営、 この高速道路上でサービスを運営するようなもの。
```
