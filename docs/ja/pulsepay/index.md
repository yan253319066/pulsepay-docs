# PulsePayとは？

## 一言で理解する PulsePay

```
PulsePay = AI使用 = マイニング

従来モデル：
AIを使用 → AI会社に支払う → 収益は会社に归属

PulsePayモデル：
AIを使用 → PulsePayに支払う → 収益の一部をあなたに（AIP保有者）
                        ↑
                        AIP保有 = AI発展の配当금을共有
```

## コアポジション

**PulsePay Protocol**は、AI使用行動をチェーン分配可能収益に変換する最初期インフラプロトコルです。

## コア優勢

### 1. 実際のビジネス収益が駆動

```
❌ 従来のDeFiマイニング：
収益 = 新規ユーザーの資金
本質 = ポンジースキーム
新規資金が止まると = 収益がゼロに

✅ PulsePay：
収益 = AI API手数料 + 決済手数料
本質 = 実際のビジネス収益
ユーザー増加 = プラットフォーム収益増加 = 収益増加
```

### 2. 完全チェーン上透明

```
チェーン上で全データ検証可能：
- プラットフォーム日次収益 → 公開検証可能
- 全 AIPステーク量 → 公開検証可能
- 収益分配記録 → 公開検証可能

検証方法：
- BSCScanでコントラクトを確認
- 自分の予想収益を計算
```

### 3. プログレッシブ分散化

```
Phase 1：インフラ構築 ✅
Phase 2：Token発行 ✅
Phase 3：収益分配 ✅（現在）
Phase 4：エコシステム拡張
Phase 5：完全DAO
```
