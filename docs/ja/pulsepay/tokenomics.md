## Phase 1 交換メカニズム

### ルール

```
交換価格：1 USDT = 1 AIP（固定）

サポート資産：
- USDT（BSC/TRC-20/ERC-20）
- BNB（リアルタイム為替レートで換算）

最低交換：10 USDT
最高交換：10,000 USDT/人

退出方式：
- Phase 1ではDEXで売却不可
- 収益分配に参加可能
- Phase 2 開放後にDEXで取引可能
```

## 通縮メカニズム

### バイバック销毁

```
30% のプラットフォーム収益用于バイバック销毁

メカニズム：
1. 毎日USDTで市場からAIPをバイバック
2. AIPを销毁アドレスに送信（永久移除）
3. 销毁されたAIPは永久に 시장에서消滅

効果：
- 流通量减少
- 供給減少
- 価値支撑
```
