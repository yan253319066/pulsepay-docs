# 什麼是 Web3？

## 一句話理解 Web3

**Web3 = 去中心化的互聯網**

你可以把互聯網想象成一座城市：

| 版本 | 特點 | 類比 |
|------|------|------|
| **Web1** | 只讀 | 紙質報紙（只能看） |
| **Web2** | 讀寫 | 社交媒體（能看能發） |
| **Web3** | 去中心化 | 公共圖書館（每個人都是管理員） |

## Web3 的核心特徵

### 1. 🏛️ 去中心化

傳統互聯網數據存儲在一個公司的服務器上：

```
傳統模式：
用戶 → Facebook 服務器 → 用戶數據

Web3 模式：
用戶 → 區塊鏈（全球N台電腦共同維護）→ 無需信任任何中心
```

### 2. 🔐 用戶擁有數據

在 Web2 中，你的社交賬號屬於平臺：
- 平臺可以刪除你的賬號
- 平臺可以出售你的數據
- 平臺倒閉你就失去所有內容

在 Web3 中，你的數字資產屬於你自己：
- 只有你能控制你的資產
- 可以隨時轉移
- 不依賴任何平臺

### 3. 💰 原生數字支付

Web3 內置支付系統，無需銀行：
- 直接轉賬，無需中介
- 全球可用，秒級到賬
- 費用低，24/7 可用
