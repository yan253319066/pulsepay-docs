## 獲取幫助

### 問題回饋

```
遇到問題？

1. 先查看 FAQ
   本文件的常見問題部分

2. 搜尋社區
   Discord/Telegram 搜尋關鍵字

3. 提問
   在社區相應頻道提問
   - #support 頻道
   - #questions 頻道

請提供：
- 問題描述
- 截圖
- 交易 TxID（如適用）
```

### 回報問題

```
發現 Bug 或安全問題？

1. 一般 Bug
   - Discord #bugs 頻道
   - 詳細描述復現步驟

2. 安全問題
   - 發送到 security@pulsepay.fun
   - 保持謹慎，不公開細節

3. 智能合約漏洞
   - Bug Bounty 計畫
   - 詳情見安全頁面
```
