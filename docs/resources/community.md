```

1. 先查看 FAQ

   Discord/Telegram 搜索关键词

3. 提问
   在社区相应频道提问
   - #support 频道
   - #questions 频道

- 问题描述
```

### 报告问题

```

1. 普通 Bug
   - Discord #bugs 频道
   - 详细描述复现步骤

2. 安全问题
   - 发送到 security@pulsepay.fun
   - 保持谨慎，不公开细节

3. 智能合约漏洞
   - Bug Bounty 计划
   - 详情见安全页面
```

## 社区规则

### 行为准则

```
✅ 鼓励：
- 友善交流
- 帮助新用户
- 分享知识
- 建设性讨论

❌ 禁止：
```
