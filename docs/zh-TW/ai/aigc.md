# AIGC入門

## 什麼是AIGC？

```
AIGC = AI Generated Content = AI生成內容

對比：
- PGC：專業生成內容
  例：記者寫的新聞
  
- UGC：用戶生成內容
  例：用戶發的視頻

- AIGC：AI生成內容
  例：AI寫的文章、AI畫的圖
```

## AIGC能生成什麼？

### 📝 文字內容

```
1. 文章寫作
   - 新聞報道
   - 產品文案
   - 社交媒體帖子

2. 代碼生成
   - 寫代碼
   - 代碼審查
   - 技術文檔

3. 創意寫作
   - 小說、故事
   - 詩歌
   - 劇本
```

### 🎨 圖像生成

```
1. 插畫/配圖
   - Midjourney
   - DALL-E 3
   - Stable Diffusion

2. 海報設計
   - 營銷素材
   - 活動海報
   - 產品展示

3. 頭像/人物
   - AI頭像
   - 遊戲角色
   - 虛擬偶像
```
