# DeFi 是什么？

## 一句话理解 DeFi

**DeFi = Decentralized Finance = 去中心化金融**

把传统银行的功能，用代码搬到区块链上：
- 存钱 → 获得利息
- 借钱 → 支付利息
- 交易 → 兑换资产
- 理财 → 获得收益

```
传统银行：
你 → 存钱到银行 → 银行借给别人 → 银行赚差价 → 给你一点利息
   ↓
你的隐私被银行掌握
你的钱受银行控制

DeFi：
你 → 存钱到智能合约 → 智能合约自动借出 → 算法决定利率 → 收益归你
   ↓
代码即法律，无须信任
你的钱在区块链上，合约里
```

## DeFi vs 传统金融

| 维度 | 传统金融 | DeFi |
|------|----------|------|
| **准入门槛** | 需要身份验证、银行账户 | 只需要钱包地址 |
| **服务时间** | 工作日 9:00-17:00 | 24/7 全天候 |
| **地域限制** | 跨境困难 | 全球通用 |
| **透明度** | 黑箱操作 | 代码开源，链上可查 |
| **收益率** | 年化 1-5% | 年化 5-100%+ |
| **控制权** | 第三方托管 | 自己掌控资产 |

## DeFi 的核心应用

### 1. 💰 借贷 (Lending/Borrowing)

**存款赚利息：**
```
传统银行：
活期存款利率 ≈ 0.3%/年

DeFi（Aave/Compound）：
稳定币存款利率 ≈ 3-8%/年
```

**抵押借款：**
```
1. 存入 1000 USDT 作为抵押
2. 最多可借 700 USDT（70% LTV）
3. 支付利息还款
4. 取回抵押物
```

### 2. 🔄 交易 (DEX - 去中心化交易所)

**交易所对比：**

```
中心化交易所（币安）：
- 你把钱存给交易所
- 交易所记账
- 需要 KYC 身份认证

去中心化交易所（Uniswap/PancakeSwap）：
- 你用钱包直接交易
- 智能合约自动撮合
- 无需注册，匿名交易
```

### 3. 📈 收益聚合 (Yield Farming)

```
简单理解：把钱存入 DeFi 协议
        ↓
        自动寻找最高收益的策略
        ↓
        复利收益最大化
```

### 4. 🎯 质押 (Staking)

```
把钱锁在协议里
↓
获得验证区块的奖励
↓
年化收益 5-20%
```

## DeFi 术语表

| 术语 | 解释 | 类比 |
|------|------|------|
| **TVL** | 总锁仓价值 | 银行的"存款总额" |

| **LP** | 流动性提供者 | 股市的"庄家" |
| **无常损失** | 提供流动性时价格变动造成的损失 | — |

## DeFi 风险

### ⚠️ 智能合约风险
```
代码漏洞可能导致资金损失
防范：选择经过审计的知名协议
```

### ⚠️ 无常损失 (Impermanent Loss)
```
当你在 DEX 提供流动性时
如果代币价格大幅波动
你获得的收益可能不如单纯持有
```

### ⚠️  rug pull（跑路）
```
项目方卷走资金
防范：查看代码是否开源、团队是否知名
```

### ⚠️ 清算风险
```
抵押物价值下跌
可能触发强制清算
防范：保持足够的抵押率
```
