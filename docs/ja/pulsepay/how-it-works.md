
## 収益分配メカニズム

### 収益源

```
AI Gateway収益 = AI呼び出し手数料
決済システム収益 = 決済処理手数料

每次の取引：
- ユーザーが100 USDTを支払う
- プラットフォームが手配料1 USDTを收取 (1%)
- この1 USDTが「収益」
```

### 分配割合

```
毎日のプラットフォーム収益分配：

┌────────────────────────────────────────┐
│            100% プラットフォーム収益               │
├────────────────────────────────────────┤
│                                        │
│   40% ─────────────────→ AIP ステーク者配当 │
│                                        │
│   30% ─────────────────→ バイバック销毁 AIP │
│                                        │
│   20% ─────────────────→ チーム運営       │
│                                        │
│   10% ─────────────────→ エコシステム激励 │
│                                        │
└────────────────────────────────────────┘
```

## 参加方法

### ステップ1：準備作業

```
必要なツール：
1. 暗号ウォレット（MetaMaskを推奨）
2. 少量のBNB（Gas手数料用）
3. USDT（ AIP と交換用）
```

### ステップ2：AIPを取得

```
方式一：公式交換
1. stake.pulsepay.funにアクセス
2. ウォレットを接続
3. USDT/BNBでAIPを交換
4. Phase 1 固定価格：1 USDT = 1 AIP

方式二：二次市場
- Phase 2 開放後
- DEX で取引可能
```

### ステップ3：质押して参加

```
1. stake.pulsepay.funにアクセス
2. ウォレットを接続
3. AIPを质押
4. 最低10 AIPから
5. 毎日収益を獲得開始
```
