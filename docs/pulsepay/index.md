# PulsePay 是什么？

## 一句话理解 PulsePay

```
PulsePay = AI 使用付费 + 激活 AIP 收益分成 获得收益分成

传统模式：
你使用 AI → 付费给 AI 公司 → 收益归公司

PulsePay 模式：
你使用 AI → 付费给 PulsePay（USDT/BNB）
                        ↓
                  平台获得真实业务收入
                        ↓
        激活 AIP 收益分成 → 参与收益分成 → 每日获得 USDT/BNB 分红
```

## 核心定位

**PulsePay Protocol** 是首个将 AI 使用行为转化为链上可分配收益的基础设施协议。

![PulsePay AI 使用流程](/images/pulsepay/ai-mining.svg)

## 解决了什么问题？

### AI 用户的痛点

| 痛点 | 传统方案 | PulsePay 方案 |
|------|----------|---------------|
| **多平台账号** | OpenAI、Anthropic、Google...分别注册 | 一个账号，统一接入 |
| **计费分散** | 每个平台单独充值、结算 | USDT/BNB 统一支付 |
| **使用无回报** | 付费使用，仅此而已 | 激活 AIP 收益分成 获得分红 |

### AIP 持有者的机会

| 传统 | PulsePay |
|------|----------|
| 买币等升值 | 质押 = 每日获得 USDT/BNB 分红 |
| 不知道有什么价值 | AIP = 平台使用权的证明 |
| 只能靠卖币赚钱 | 使用 + 分红双重收益 |


## 常见问题

### Q：AIP 会涨吗？

A：价格由市场决定。但 PulsePay 有：
- 30% 收入回购销毁（通缩）
- 真实业务收入支撑（价值支撑）
- 使用场景（实用价值）

### Q：收益有多少？

A：收益取决于：
- 平台每日业务收入
- 你的 AIP 质押量
- 全网质押总量

### Q：安全吗？

A：PulsePay 已实施：
- 第三方安全审计
- 时间锁机制
- 多签控制
- 渐进式去中心化
