# DAO 組織

## 什麼是 DAO？

```
DAO = Decentralized Autonomous Organization
    = 去中心化自治組織

類比：
- 傳統公司：老闆/管理層決策
- DAO：代碼 + 代幣持有者決策

特點：
- 沒有中心化領導
- 規則寫在智能合約裡
- 成員共同治理
```

## DAO vs 傳統組織

| 維度 | 傳統公司 | DAO |
|------|----------|-----|
| 決策方式 | 管理層決定 | 社區投票決定 |
| 所有權 | 股東北持股 | Token 持有者 |
| 規則執行 | 人為執行 | 代碼自動執行 |
| 透明程度 | 內部資訊不透明 | 鏈上公開透明 |
| 準入門檻 | 招聘/入職 | 購買 Token |
| 退出方式 | 辭職/辭退 | 出售 Token |
| 地理限制 | 有 | 無 |

## DAO 的工作原理

### 1. Token 治理

```
治理代幣 = 投票權

持有代幣 = 獲得投票權
投票權重 = 代幣數量

示例：
- 持有 1000 AIP → 1000 票
- 持有 10000 AIP → 10000 票
```

### 2. 提案系統

```
提案流程：

1. 創建提案
   任何人可以創建（有時需要最低代幣門檻）

2. 討論期
   社區討論、辯論
   通常 2-7 天

3. 投票期
   Token 持有者投票
   通常 3-5 天

4. 執行期
   提案通過 → 自動執行
   提案失敗 → 放棄
```

## DAO 與 PulsePay

### 漸進式去中心化

```
當前階段（Phase 1-3）：
- 團隊主導開發
- 收集社區反饋
- 逐步開放治理

未來目標（Phase 4-5）：
- 開放完全治理
- 社區主導決策
- DAO 化
```

### 治理路線圖

```
Phase 3：開始社區治理
- 收集社區提案
- 論壇討論

Phase 4：開放投票
- 開啟鏈上投票
- 社區決定參數

Phase 5：完全 DAO
- 團隊代幣解鎖完成
- 真正去中心化
```

如何參與 PulsePay 治理：

2. 加入社區

3. 提交提案
   - 提出改進建議
   - 參與討論

4. 投票
   - 對提案投票
   - 參與重要決策
