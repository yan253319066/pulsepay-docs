## ヘルプを取得

### 問題フィードバック

```
問題に遭遇？

1. まず FAQ を確認
   このドキュメントのよくある質問セクション

2. コミュニティを検索
   Discord/Telegram でキーワードを検索

3. 質問する
   コミュニティの対応チャンネルで質問
   - #support チャンネル
   - #questions チャンネル

必ず提供：
- 問題の説明
- スクリーンショット
- 取引 TxID（該当する場合）
```

### 問題報告

```
Bug またはセキュリティ問題を 발견？

1. 通常の Bug
   - Discord #bugs チャンネル
   - 再現手順を詳しく説明

2. セキュリティ問題
   - security@pulsepay.fun に送信
   - 慎重に、詳細は公開しない

3. スマートコントラクトの脆弱性
   - Bug Bounty プログラム
   - 詳細はセキュリティページを参照
```

## 参加と貢献

### 貢献方法

```
1. コンテンツ創作
   - チュートリアルを作成
   - 動画を制作
   - ドキュメントを翻訳
   - 使用経験を分享

2. 技術貢献
   - ツールを開発
   - コードを記述
   - ドキュメントを整備
   - セキュリティ監査

3. コミュニティ貢献
   - 新規ユーザーを支援
   - アクティビティを組織
   - フィードバックを提供
```

## コミュニティルール

### 行動規範

```
✅ 奨励：
- フレンドリーな交流
- 新規ユーザーを支援
- 知識を分享
- 建設的な議論

❌ 禁止：
- 人身攻撃
- スパム
- 詐欺またはフィッシング
- 他人のプライバシー泄露
- 政治/宗教議論
```

### 違反処理

```
違反処理方法：
- 警告
- ミュート
- グループからキック
- ブラックリストに登録

重大な違反：
- 関連機関に報告
- 法的訴追の権利を留保
```
