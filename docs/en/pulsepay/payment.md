## Recharge Guide

### Step 1: Get Recharge Address

```
1. Login app.pulsepay.fun
2. Click "Recharge"
3. Select coin and network
4. Copy recharge address

⚠️ Note:
- Must select correct network!
- USDT TRC-20 ≠ USDT ERC-20
- Wrong network = Asset permanently lost!
```

### Step 2: Initiate Transfer

```
From exchange withdrawal:

1. Login exchange (Binance/OKX)
2. Go to "Withdrawal" page
3. Select coin (USDT)
4. Select network (must match recharge page!)
5. Paste recharge address
6. Enter withdrawal amount
7. Confirm withdrawal
```

### Step 3: Wait for Confirmation

```
Confirmation process:

TRC-20 recharge: ~1 minute
ERC-20 recharge: ~5-15 minutes
BEP-20 recharge: ~1 minute

Check progress:
- Query on blockchain explorer
- Wait for 1-3 block confirmations
```

## Withdrawal Guide

### Step 1: Initiate Withdrawal

```
1. Login app.pulsepay.fun
2. Click "Withdraw"
3. Select coin and network
4. Enter target address
5. Enter withdrawal amount
```

### Step 2: Confirm Information

```
Pre-withdrawal confirmation:

□ Coin correct
□ Network correct (BSC/TRON/ETH)
□ Address correct
□ Amount correct
□ Network fee reasonable

Network fees:
- TRC-20: 1 USDT
- ERC-20: ~5-10 USDT
- BEP-20: ~0.5 USDT
```

## Security Measures

### Cold/Hot Wallet Separation

```
Hot wallet:
- Daily operations
- Limit controls
- Online storage

Cold wallet:
- Large asset storage
- Fully offline
- Multi-sig management
```

### Real-time Monitoring

```
System monitoring:
- Anomalous transaction alerts
- Large transfer review
- Risk control

Risk control measures:
- Blacklist addresses
- Suspicious behavior detection
- Manual review
```
