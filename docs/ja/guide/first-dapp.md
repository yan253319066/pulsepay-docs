# 初めてDAppを使う

## DAppとは？

```
DApp = Decentralized Application = 分散型アプリケーション

例え：
App = 手机にインストールしたアプリ（微信、抖音）
     ↓
Appのデータは会社のサーバーに保存

DApp = ブラウザ内のアプリ
     ↓
DAppのデータはブロックチェーンに保存
```

## 準備作業

### 必要なツール

```
1. ブラウザウォレット（MetaMask推奨）
   👉 metamask.io

2. 少量の加密通貨（Gas手数料用）
   推奨：BNB（BSCチェーン用）

3. ステーブルコイン（USDT）取引用
   推奨：まずは少額で尝试
```

### ステップ1：MetaMaskをインストール

```
1. metamask.ioにアクセス
2. 「Download」をクリック
3. お好みのブラウザを選択
4. プラグインをインストール
5. 「開始する」をクリック
6. 「新しいウォレットを作成」を選択
7. パスワードを設定
8. シードフレーズ（12単語）を保存
```

## PulsePayを使用

### ステップ1：PulsePayにアクセス

```
1. ブラウザを開く
2. app.pulsepay.funにアクセス
3. 「ウォレット接続」をクリック
4. MetaMaskを選択
5. 署名を確認
```

### ステップ2：充值

```
1. PulsePayウォレットページで
2. 「充值」をクリック
3. USDT + BSCネットワークを選択
4. 充值アドレスをコピー
5. 取引所からこのアドレスに引出
```

### ステップ3：収益分配に参加

```
1. stake.pulsepay.funにアクセス
2. ウォレットを接続
3. AIPトークンを质押
4. 取引を確忍
5. 每日収益を獲得開始
```
