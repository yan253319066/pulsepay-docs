## 通缩机制

### 回购销毁

```
30% 的平台收入用于回购销毁

机制：
1. 每日用 USDT 从市场回购 AIP
2. 将回购的 AIP 发送到销毁地址
3. 销毁的 AIP 永久移除

效果：
- 流通量减少
- 供给减少
- 价值支撑
```

### 销毁地址

```
黑洞地址（Burn Address）：
0x000000000000000000000000000000000000dEaD

特点：
- 没有人有私钥
- 发送到这里的代币永远无法取出
- 等同于永久销毁
```

### 通缩计算示例

```
假设：
- 每日平台收入：10,000 USDT
- 回购销毁：3,000 USDT
- AIP 价格：1 USDT

每日销毁量：3,000 AIP

年销毁量：3,000 × 365 = 1,095,000 AIP
年销毁比例：1,095,000 / 65,000,000 = 1.68%

长期看：
- 流通量持续减少
- 稀缺性增加
- 价值支撑
```

## 常见问题

### Q：为什么初始销毁 10%？

A：初始销毁是为了：

A：不会。AIP 是固定总量，没有增发机制。实际会发生通缩（回购销毁）。

### Q：Phase 1 没来得及怎么办？

A：Phase 2 开放后可以在 DEX 购买，但价格将由市场决定，可能高于或低于 1 USDT。

### Q：团队代币解锁会砸盘吗？

A：不会，因为：
- 有 1 年锁仓

- 团队与项目长期绑定
