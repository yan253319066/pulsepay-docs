# 常见问题 (FAQ)

## 关于 PulsePay

### PulsePay 是什么？

**PulsePay Protocol** 是首个将 AI 使用行为转化为链上可分配收益的基础设施协议。通过整合 AI Gateway、加密支付系统与数字钱包，构建了一套完整的"AI 收费 → 收入沉淀 → Token 分配"闭环经济体系。

### PulsePay 合法吗？

PulsePay 是一个技术协议，核心功能是：
- 提供 AI 调用服务
- 提供支付服务
- 提供收益分成服务

我们遵守运营地区的法律法规，并采取合规措施。

### PulsePay 团队是谁？

团队信息将在 Phase 3 阶段逐步公开。目前采取以下信任建立措施：
- 产品已上线可验证
- 合约代码开源
- 收入链上透明
- 渐进式去中心化

## 关于 AIP Token

### AIP 是什么？

AIP (AI Pulse) 是 PulsePay 协议的治理代币，持有者可以：
- 参与收益分成
- 分享平台业务增长红利
- 未来参与协议治理

### AIP 总供应量是多少？

100,000,000 AIP（1 亿枚），固定总量，无增发机制。

### AIP 的价值来源是什么？

AIP 有多重价值支撑：
1. **使用价值** - 质押参与收益分成
2. **通缩机制** - 30% 收入回购销毁
3. **需求驱动** - 用户需要 AIP 参与生态

### Phase 1 没来得及怎么办？

Phase 2 开放后可以在 DEX（如 PancakeSwap）购买 AIP，届时价格将由市场供需决定。

## 关于收益

### 收益从哪里来？

收益来自 PulsePay 平台的真实业务收入：
- AI Gateway 调用手续费
- 支付系统处理手续费

不是旁氏结构，不是通胀激励。

### 收益是多少？

收益取决于：
- 每日平台业务收入
- 你的 AIP 质押量
- 全网 AIP 质押总量

计算公式：
```
个人收益 = (个人质押量 / 全网质押量) × (当日平台收入 × 40%)
```

### 收益怎么领取？

需要手动点击"领取"按钮，收益会转入你的钱包。

### 收益需要纳税吗？

取决于你所在国家/地区的税务法规。建议咨询当地税务顾问。

## 关于质押

### 最低质押多少？

最低 10 AIP 起。

### 有锁定期吗？

没有锁定期，可以随时质押、随时解除。

### 解除质押会影响已领取的收益吗？

不影响。已领取的收益归你所有，解除质押只影响未来的新收益。

### 质押有风险吗？

主要风险：
- AIP 价格波动
- 平台业务下降

不存在：
- 爆仓风险
- 清算风险
- 惩罚机制

## 关于安全

### PulsePay 安全吗？

PulsePay 已实施多重安全措施：
- ✅ 第三方安全审计
- ✅ 时间锁机制

### 私钥丢失了怎么办？

**无法找回！** 加密货币没有"找回密码"功能。请务必：
- 备份助记词
- 多份保存
- 防水防火

### 如何防止被骗？

1. **永远不要告诉任何人私钥**
2. **只访问官方链接**
3. **仔细核对网址**
4. **大额转账前先小额测试**
