# Web3とは？

## 一言で理解する Web3

**Web3 =分散型インターネット**

インターネットを一つの都市に例えると：

| バージョン | 特徴 | たとえ |
|------------|------|--------|
| **Web1** | 読み取り専用 | 新聞（読めるだけ） |
| **Web2** | 読み書き | SNS（読んで投稿できる） |
| **Web3** | 分散型 | 図書館（全員が管理者） |

## Web3のコア機能

### 1. 🏛️ 分散型

従来のインターネットデータは一つの企業のサーバーに保存されます：

```
従来モデル：
ユーザー → Facebookサーバー → ユーザーデータ

Web3モデル：
ユーザー → ブロックチェーン（世界中のN台のPCで維持）→ 中央を信頼する必要なし
```

### 2. 🔐 ユーザーがデータを所有

Web2では、SNSアカウントはプラットフォームに属します：
- プラットフォームはアカウントを削除できる
- プラットフォームはデータを出售できる
- プラットフォームが倒闭すると全て失う

Web3では、デジタル資産は自分にあります：
- 自分が資産をコントロール
- いつでも转移可能
- どのプラットフォームにも依存しない

### 3. 💰 原生デジタル決済

Web3には決済システムが組み込まれています：
- 銀行なしで直接送金
- 世界中で利用可能、即座に決済
- 手数料が安い、24時間365日利用可能
