# AIGC 入门

## 什么是 AIGC？

```
AIGC = AI Generated Content = AI 生成内容

对比：
- PGC：Professional Generated Content（专业生成）
  例：记者写的新闻
  
- UGC：User Generated Content（用户生成）
  例：用户发的抖音视频

- AIGC：AI Generated Content（AI 生成）
  例：AI 写的文章、画的图
```

## AIGC 的使用技巧

### 1. 提示词 (Prompt) 技巧

```
❌ 差提示词：
"画一幅画"

✅ 好提示词：
"画一幅赛博朋克风格的城市夜景
霓虹灯闪烁
有飞行汽车
比例 16:9
电影感灯光"
```

### 2. 结构化输入

```
好的输入结构：

1. 任务说明
   "你是一个资深产品经理"

2. 背景信息
   "我们是一个面向 Z 世代的咖啡品牌"

3. 具体要求
   "写 3 条小红书文案，每条 100 字"

4. 输出格式
   "用 emoji 分隔"
```

### 3. 迭代优化

```
第一轮：AI 生成初稿
  ↓
第二轮：根据初稿提出修改意见
  ↓
第三轮：AI 修改
  ↓
第四轮：人工润色
  ↓
完成
```

## AIGC 的挑战和风险

### ⚠️ 版权问题

```
问题：
- AI 生成的图像可能有版权争议
- 训练数据来源不明确

解决方案：
- 使用明确授权的工具
- 保留生成记录
- 企业使用商业安全版本
```

### ⚠️ 质量控制

```
问题：
- AI 生成内容可能有问题
- 需要人工审核

解决方案：
- 建立审核流程
- 重要内容人工复核
- 持续优化提示词
```

### ⚠️ 滥用风险

```
问题：
- 深伪技术 (Deepfake)
- 虚假新闻
- 抄袭洗稿

解决方案：
- 技术手段检测
- 内容溯源
- 平台监管
```
