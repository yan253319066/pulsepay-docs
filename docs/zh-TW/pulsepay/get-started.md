# 快速開始指南

## 準備工作

### 需要準備

```
1. 加密錢包
   推薦：MetaMask
   安裝：瀏覽器插件或手機 App

2. 少量加密貨幣
   BNB：約 0.01 BNB（支付 Gas 費）
   USDT：用於兌換 AIP（最低 10 USDT）

3. 一個交易所賬號
   用於購買 USDT/BNB
   推薦：幣安、OKX
```

### 第一步：獲取錢包

**方案一：MetaMask（瀏覽器插件）**

1. 訪問 [MetaMask](https://metamask.io)
2. 下載瀏覽器插件
3. 創建錢包
4. 備份助記詞（12個單詞）
5. 設置密碼

**方案二：MetaMask（推薦）**

MetaMask 是最流行的 Web3 錢包，可以完全支持 PulsePay 的所有功能。

### 第二步：配置 BSC 網絡

MetaMask 添加 BSC 網絡：

1. 點擊 MetaMask 插件
2. 點擊網絡選擇器
3. 點擊「添加網絡」
4. 填寫以下信息：

   - 網絡名稱：BSC
   - RPC URL：`https://bsc-dataseed.binance.org/`
   - 鏈 ID：56
   - 符號：BNB
   - 區塊瀏覽器：[https://bscscan.com](https://bscscan.com)

5. 點擊保存

### 第三步：獲取 USDT

**方式一：交易所購買**

1. 登錄幣安/OKX
2. 進入 C2C 交易
3. 買入 USDT（選擇 BNB 網絡）
4. 提現到你的錢包地址

::: warning ⚠️ 注意
- 網絡選擇 BSC (BEP-20)
- 提現地址填寫你的錢包地址
- 小額測試確認無誤後再大額
:::

**方式二：別人轉賬**

從其他錢包轉賬 USDT (BSC) 到你的地址

## 參與 Phase 1

### 兌換 AIP

1. 訪問 [stake.pulsepay.fun](https://stake.pulsepay.fun)
2. 點擊「連接錢包」
3. 選擇 MetaMask
4. 授權連接
5. 進入「兌換」頁面
6. 輸入 USDT 數量
   - 最低 10 USDT
   - 最高 10,000 USDT/人
7. 確認兌換
   - 1 USDT = 1 AIP
   - 無額外手續費
8. 等待鏈上確認
9. 完成！AIP 已到賬

### 啟動 AIP 收益分成

1. 進入「質押」頁面
2. 輸入要質押的 AIP 數量
3. 點擊「質押」
4. 確認交易
5. 等待鏈上確認

::: tip ⚠️ 最低質押：10 AIP
完成！從明天開始獲得每日收益
:::

## 領取收益

### 查看收益

1. 進入「收益」頁面
2. 查看以下數據：
   - 累計收益
   - 今日收益
   - 待領取收益

**收益計算**：
- 每日 UTC 0 點結算
- 按質押比例分配

### 領取收益

1. 點擊「領取」按鈕
2. 確認領取數量
3. 點擊確認
4. 等待鏈上確認
5. USDT/BNB 到賬錢包

::: warning ⚠️ 注意
- 收益需要手動點擊「領取」
- 不會自動到賬
- 建議每週或每天領取一次
:::
