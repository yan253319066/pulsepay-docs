# 收益分成详解

## 什么是收益分成？

```
收益分成 = Revenue Share = 分享平台收入

持有 AIP → 质押参与 → 每日获得 USDT/BNB 分红

类比：
持有苹果股票 → 获得苹果公司分红
激活 AIP 收益分成 → 获得 PulsePay 平台分红

区别：
- 股票分红来自利润
- AIP 分红来自平台业务收入
```

## 如何参与？

### 参与条件

```
最低门槛：10 AIP

无锁定期：随时可质押/解除
无最高限制：想质押多少都可以

⚠️ 注意：
- 质押越多，分红越多
- 解质押需要等待
- 收益需要手动领取
```

### 质押步骤

```
1. 访问 stake.pulsepay.fun
2. 连接钱包（MetaMask/PulseWallet）
3. 进入"质押"页面
4. 输入 AIP 数量（或点击最大值）
5. 点击"质押"
6. 确认交易
7. 等待链上确认
8. 完成！开始获得每日收益
```

### 领取收益

```
收益领取：

1. 进入"收益"页面
2. 查看累计收益
3. 点击"领取"
4. 收益转入钱包

⚠️ 注意：
- 收益需要手动点击领取
- 不会自动到账
- 建议定期领取
```

### 解除质押

```
解除质押：

1. 进入"质押"页面
2. 输入解除数量
3. 点击"解除质押"
4. 确认交易

⚠️ 注意：
- 解除后不再获得新收益
- 已领取的收益不会收回
- 解除需要链上确认
```

## 收益计算

### 计算公式

```
个人每日收益 = (个人质押量 / 全网质押总量) × (当日平台收入 × 40%)
```

### 计算示例

```
假设条件：
- 今日平台收入：10,000 USDT
- 收益分成比例：40%
- 你的质押量：10,000 AIP
- 全网质押总量：1,000,000 AIP

计算过程：
1. 可分配收益 = 10,000 × 40% = 4,000 USDT
2. 你的份额 = 10,000 / 1,000,000 = 1%
3. 你的收益 = 4,000 × 1% = 40 USDT

结论：质押 10,000 AIP，每日获得 40 USDT
```

### 年化收益计算

```
年化收益率计算：

日收益：40 USDT
年收益：40 × 365 = 14,600 USDT
投资本金：10,000 AIP（当时价值 10,000 USDT）
年化收益：14,600 / 10,000 = 146%

⚠️ 注意：
- 这是假设每日收入不变的情况
- 实际收入会波动
- 这是预期收益，不是保证收益
```

## 常见问题

### Q：收益是固定的吗？

A：不是固定的。收益取决于：
- 每日平台收入
- 质押量占全网比例
- 业务发展情况

### Q：需要每天领取吗？

A：建议定期领取，但：
- 收益会累积，不会过期
- 可以一次性领取所有历史收益
- 领取是免费的（无额外 Gas 费）

### Q：解除质押会影响已领取的收益吗？

A：不影响。
- 已领取的收益归你
- 解除质押只影响未来的新收益

### Q：质押有什么风险？

A：主要风险：
- AIP 价格波动
- 平台业务下降

不存在：
- 不会爆仓
- 没有清算
- 没有惩罚

### Q：收益需要纳税吗？

A：取决于你所在国家的法律。建议咨询当地税务顾问。
