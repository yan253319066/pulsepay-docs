# 収益分配の詳細

## 収益分配とは？

```
収益分配 = Revenue Share = プラットフォーム収益の共有

AIP保有 → ステーク → 毎日USDT/BNB配当受領

例え：
苹果株保有 → Apple会社の配当を獲得
AIP保有 → PulsePayプラットフォームの配当を獲得

違い：
- 株の配当は利益から
- AIPの配当はプラットフォーム事業収益から
```

```

```

## 参加方法

### ステーク手順

```
1. stake.pulsepay.funにアクセス
2. ウォレット接続（MetaMask）
3. 「ステーク」ページに移動
4. AIP数量を入力（または最大）
5. 「ステーク」をクリック
6. 取引を確認
7. チェーン確認を待機
8. 完了！毎日収益を獲得開始
```

### 収益受領

```
1. 「収益」ページに移動
2. 累計収益を確認
3. 「受け取る」をクリック
4. 収益がウォレットに着金

⚠️ 注意：
- 収益は手動で「受け取る」が必要
- 自動では着金しない
- 定期的に受け取ることを推奨
```

## 収益計算

### 計算式

```
個人每日収益 = (個人ステーク量 / 全ネットステーク量) × (当日プラットフォーム収益 × 40%)
```

### 計算例

```
前提条件：
- 当日プラットフォーム収益：10,000 USDT
- 収益分配比率：40%
- あなたのステーク量：10,000 AIP
- 全ネットステーク量：1,000,000 AIP

計算：
1. 分配可能額：10,000 × 40% = 4,000 USDT
2. あなたの割合：10,000 / 1,000,000 = 1%
3. あなたの収益：4,000 × 1% = 40 USDT

結論：10,000 AIPで収益分配を起動、毎日40 USDTを獲得
```

### 年利計算

```
日次収益：40 USDT
年収益：40 × 365 = 14,600 USDT
投資元本：10,000 AIP（当時10,000 USDT相当）
年利：14,600 / 10,000 = 146%

⚠️ 注意：
- 毎日収益が一定と仮定
- 実際の収益は変動
- 期待収益であり保証ではない
```
