# クイックスタートガイド

## 準備

### 必要なもの

```
1. 暗号ウォレット
   推奨：MetaMask
   インストール：ブラウザ拡張機能またはスマホアプリ

2. 少量の暗号通貨
   BNB：約0.01 BNB（Gas手数料用）
   USDT：AIP交換用（最小10 USDT）

3. 取引所アカウント
   USDT/BNB購入用
   推奨：BINANCE、OKX
```

### ステップ1：ウォレットを取得

```
方法1：MetaMask（ブラウザ拡張機能）

1. metamask.ioにアクセス
2. ブラウザ拡張機能をダウンロード
3. ウォレットを作成
4. シードフレーズ（12単語）をバックアップ
5. パスワードを設定

方法2：MetaMask（推奨）

MetaMaskは最も人気のWeb3ウォレットで、PulsePayの全機能を利用できます。

### ステップ2：BSCネットワークを設定

```
MetaMaskにBSCネットワークを追加：

1. MetaMask拡張機能をクリック
2. ネットワークセレクターをクリック
3. 「ネットワークを追加」をクリック
4. 以下を入力：

ネットワーク名：BSC
RPC URL：https://bsc-dataseed.binance.org/
チェーンID：56
シンボル：BNB
ブロックエクスプローラー：https://bscscan.com

5. 保存をクリック
```

### ステップ3：USDTを取得

```
方法1：取引所で購入

1. BINANCE/OKXにログイン
2. C2C取引に移動
3. USDTを購入（BNBネットワークを選択）
4. ウォレットアドレスに引出

⚠️ 注意：
- BSC (BEP-20)ネットワークを選択
- 引出先にウォレットアドレスを入力
- 少額をテストして問題なければ然大額へ
```

## Phase 1に参加

### AIPを交換

```
1. stake.pulsepay.funにアクセス
2. 「ウォレット接続」をクリック
3. MetaMaskを選択
4. 接続を許可

5. 「交換」ページに移動
6. USDT数量を入力
   - 最小10 USDT
   - 最大10,000 USDT/人

7. 交換を確認
   - 1 USDT = 1 AIP
   - 追加手数料なし

8. チェーン確認を待機
9. 完了！AIPがウォレットに着金
```

### AIPで収益分配を起動

```
1. 「ステーク」ページに移動
2. ステークするAIP数量を入力
3. 「ステーク」をクリック
4. 取引を確認
5. チェーン確認を待機

⚠️ 最小：10 AIP

完了！明日から日次収益を獲得
```

## 収益を受け取る

### 収益を確認

```
1. 「収益」ページに移動
2. 以下を確認：
   - 累計収益
   - 今日の収益
   - 未受領収益

収益計算：
- UTC 0時に日次決済
- ステーク比率で分配
```

### 収益を受け取る

```
1. 「受け取る」ボタンをクリック
2. 受領数量を確認
3. 確認をクリック
4. チェーン確認を待機
5. USDT/BNBがウォレットに着金

⚠️ 注意：
- 収益は手動で「受け取る」をクリック必要
- 自動では着金しない
- 毎週または毎日受け取ることを推奨
```
