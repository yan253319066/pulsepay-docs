# 第一次使用 DApp

## DApp 是什么？

```
DApp = Decentralized Application = 去中心化应用

类比：
App = 装在手机上的应用（微信、抖音）
     ↓
App的数据存在公司服务器上

DApp = 浏览器里的应用
     ↓
DApp的数据存在区块链上
```

## DApp 和普通 App 的区别

| 维度 | 普通 App | DApp |
|------|----------|------|
| **后端** | 公司服务器 | 区块链 |
| **账号** | 用户名+密码 | 钱包地址 |
| **登录** | 输入账号密码 | 连接钱包签名 |
| **数据** | 公司控制 | 用户自己控制 |
| **停服风险** | 公司倒闭就没了 | 区块链永存 |

## 准备工作

### 需要的工具

```
1. 浏览器钱包（推荐 MetaMask）
   👉 metamask.io

2. 一些加密货币（用于 Gas 费）
   推荐：BNB（用于 BSC 链）

3. 稳定币（USDT）用于交易
   建议：先少量尝试
```

### 第一步：安装 MetaMask

```
1. 访问 metamask.io
2. 点击"Download"
3. 选择你的浏览器（Chrome/Brave/Firefox）
4. 安装插件
5. 点击"开始使用"
6. 选择"创建新钱包"
7. 设置密码
8. 保存助记词（12个单词）
```

### 第二步：切换到 BSC 网络

MetaMask 默认是以太坊网络，我们需要切换到 BSC：

```
1. 打开 MetaMask
2. 点击顶部的网络选择
3. 点击"添加网络"
4. 填写 BSC 主网信息：

   网络名称：BSC
   RPC URL：https://bsc-dataseed.binance.org/
   链 ID：56
   符号：BNB
   区块浏览器：https://bscscan.com
```

## 交易签名科普

### 什么是签名？

当你连接 DApp 时，会要求你"签名"：

```
签名 = 证明"这是我的钱包"

类比：
签名 = 在文件上签你的名字
     = 证明文件是你授权的

⚠️ 重要：
签名不会转走你的资产
只是确认你拥有这个钱包
```

### 什么是授权？

有些 DApp 需要你"授权"才能操作：

```
授权 = 允许 DApp 使用你的代币

示例：
1. 你授权 AAVE 使用你的 USDT
2. AAVE 合约才能帮你借出资产

⚠️ 安全注意：
授权时可以设置额度
不要无限授权！
```

### 什么是交易？

```
交易 = 需要付 Gas 费的链上操作
     = 会真正转移资产

示例：
- 转账 USDT
- 质押代币
- 领取收益

⚠️ 交易前确认：
- 目标地址正确
- 金额正确
- 网络正确
```

## 安全检查清单

```
✅ 每次交易前检查：
□ 目标地址正确
□ 网络正确（BSC）
□ 金额正确
□ Gas 费合理

✅ 防范措施：
□ 不要点击来路不明的链接
□ 仔细核对合约地址
□ 先用小额测试
□ 不贪高收益承诺
```

## 常见问题

### Q：连接钱包安全吗？

A：安全。连接只是授权查看你的地址，不会转移资产。

### Q：签名需要付费吗？

A：一般不需要 Gas 费（除了特殊情况）。

### Q：交易失败了怎么办？

A： Gas 费还是会扣除，资产不会丢失（如果没有成功转账）。

### Q：如何取消授权？

A：可以到 https://revoke.cash/ 撤销不需要的授权。
