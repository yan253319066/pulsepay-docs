# AIGC入门

## 什么是AIGC？

```
AIGC = AI Generated Content = AI生成内容

对比：
- PGC：专业生成内容
  例：记者写的新闻
  
- UGC：用户生成内容
  例：用户发的视频

- AIGC：AI生成内容
  例：AI写的文章、AI画的图
```

## AIGC能生成什么？

### 📝 文字内容

```
1. 文章写作
   - 新闻报道
   - 产品文案
   - 社交媒体帖子

2. 代码生成
   - 写代码
   - 代码审查
   - 技术文档

3. 创意写作
   - 小说、故事
   - 诗歌
   - 剧本
```

### 🎨 图像生成

```
1. 插画/配图
   - Midjourney
   - DALL-E 3
   - Stable Diffusion

2. 海报设计
   - 营销素材
   - 活动海报
   - 产品展示

3. 头像/人物
   - AI头像
   - 游戏角色
   - 虚拟偶像
```

### 🎵 音频内容

```
1. 语音合成
   - AI配音
   - 多语言翻译
   - 克隆声音

2. 音乐生成
   - AI作曲
   - AI编曲
   - 背景音乐
```

### 🎬 视频内容

```
1. 短视频生成
   - AI虚拟主播
   - 自动剪辑
   - AI数字人

2. 动画制作
   - AI生成帧
   - 自动上色
   - 动作生成
```
