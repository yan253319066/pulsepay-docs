# NFT 完全指南

## 什麼是 NFT？

```
NFT = Non-Fungible Token = 非同質化代幣

關鍵概念：
- 同質化 (Fungible)：可以互換
  例：1美元 = 1美元，可以互換
- 非同質化 (Non-Fungible)：獨一無二
  例：蒙娜麗莎 = 只有一幅，獨一無二

NFT = 在區塊鏈上的"數位蒙娜麗莎"
```

## NFT 與傳統數位資產的區別

| 維度 | 傳統數位資產 | NFT |
|------|-------------|-----|
| 所有權證明 | 平臺內部記錄 | 區塊鏈公開可查 |
| 唯一性 | 可以無限複製 | 每個都獨一無二 |
| 可交易性 | 受平臺限制 | 可自由交易 |
| 跨平臺 | 不能 | 可以跨平臺轉移 |
| 創作者收益 | 一次性 | 持續獲得版稅 |

## NFT 的類型

### 1. 藝術收藏品

```
PFP (Profile Picture)：
- 頭像類 NFT
- 示例：CryptoPunks、Bored Ape

藝術作品：
- 數位藝術
- 生成藝術
- 示例：Art Blocks

特點：
- 收藏價值
- 社區身份
- 潛在升值
```

### 2. 遊戲道具

```
遊戲內物品：
- 武器、肌膚、角色
- 可在不同遊戲中通用
- 例：Gods Unchained

虛擬土地：
- 元宇宙土地
- 例：Decentraland、Sandbox

特點：
- 實際使用價值
- 遊戲內資產所有權
- 可交易
```

### 3. 音樂與媒體

```
音樂 NFT：
- 單曲/專輯
- 直接支持創作者
- 版稅自動分配

影片 NFT：
- 精彩時刻
- 粉絲互動
- 例：NBA Top Shot

域名 NFT：
- ENS 域名
- 數位身份
- 例：vitalik.eth
```

### 4. 實用型 NFT

```
門票/會員卡：
- 活動門票
- 俱樂部會員
- 品牌會員權益

證書/憑證：
- 學歷證明
- 所有權證明
- 數位身份

房地產：
- 鏈上房地產
- 產權證明
- 部分所有權
```

## NFT 的風險

### ⚠️ 市場風險

```
NFT 價格波動極大：
- 可能一天漲跌 50%+
- 流動性差
- 難以變現

防範：
- 只投資你能承受損失的錢
- 不要 FOMO（錯失恐懼）
- 做好研究再購買
```

### ⚠️ 版權風險

```
問題：
- 買了 NFT ≠ 擁有版權
- 版權仍在創作者手中
- 不能商用

防範：
- 仔細閱讀 NFT 條款
- 確認版權歸屬
```

### ⚠️ Rug Pull 風險

```
問題：
- 項目方卷款跑路
- NFT 價格歸零

防範：
- 調查團隊背景
- 查看合約是否開源
- 社區是否活躍
```
