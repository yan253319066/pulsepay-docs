# 暗号通貨入門

## 暗号通貨とは？

ブロックチェーン技術に基づくデジタル資産で、いかなる政府や銀行にも管理されません。

```
従来通貨：
米ドル/人民元 → 政府信用 → 銀行会計 → あなたが保有

暗号通貨：
BTC/ETH/USDT → コード即法律 → ブロックチェーン会計 → あなたが保有（秘密鍵経由）
```

## 主流暗号通貨

### 🟠 ビットコイン (BTC) - デジタルゴールド

```
創設者：サトシ・ナカモト（匿名）
launch：2009年
総供給量：2100万枚（上限あり）
位置づけ：デジタルゴールド、価値保存
```

### 🔷 イーサリアム (ETH) - デジタルオイル

```
創設者：ヴィタリック・ブテリン
launch：2015年
総供給量：上限なし（burn機構あり）
位置づけ：スマートコントラクトプラットフォーム
```

### 📝 ステーブルコイン (USDT/USDC)

```
特徴：米ドルにペッグ
1 USDT ≈ 1 米ドル
用途：価値保存、決済媒質
```

## ウォレット基礎

### 🔑 秘密鍵 vs 公開鍵 vs アドレス

```
秘密鍵
   ↓ 暗号化
公開鍵
   ↓ ハッシュ
アドレス = あなたの銀行口座
```

**重要**：
- 秘密鍵 = 銀行パスワード（絶対に秘密！）
- 公開鍵 = 銀行口座番号（共有可能）
- アドレス = 受取口座（共有可能）
