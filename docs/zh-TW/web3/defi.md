# DeFi 是什麼？

## 一句話理解 DeFi

**DeFi = Decentralized Finance = 去中心化金融**

將傳統銀行功能移到區塊鏈上，用代碼實現：
- 存錢 → 賺利息
- 借錢 → 支付利息
- 交易 → 交換資產
- 投資 → 賺取回報

```
傳統銀行：
你 → 存款到銀行 → 銀行借給別人 → 銀行賺差價 → 你拿微薄利息

DeFi：
你 → 存款到智能合約 → 智能合約自動借出 → 演算法決定利率 → 回報給你
```

## DeFi vs 傳統金融

| 維度 | 傳統 | DeFi |
|------|------|------|
| **準入門檻** | 需要身份驗證、銀行帳戶 | 只需要錢包地址 |
| **服務時間** | 週一至週五 9:00-17:00 | 24/7 |
| **地理限制** | 跨境困難 | 全球通用 |
| **透明度** | 黑盒子 | 開源，鏈上可驗證 |
| **收益** | 1-5% 年化 | 5-100%+ 年化 |
| **控制權** | 第三方保管 | 自己保管 |

## 核心 DeFi 應用

### 1. 💰 借貸/借款

**賺取利息：**
```
傳統銀行：
存款利率 ≈ 0.3%/年

DeFi (Aave/Compound)：
穩定幣存款利率 ≈ 3-8%/年
```

### 2. 🔄 交易 (DEX)

```
中心化交易所 (Binance)：
- 你把錢存到交易所
- 交易所記錄
- 需要 KYC

DEX (Uniswap/PancakeSwap)：
- 直接用錢包交易
- 智能合約自動撮合
- 匿名，無需註冊
```

### 3. 📈 收益農業

```
簡單理解：把錢放到 DeFi 協議
        ↓
        自動尋找最高收益策略
        ↓
        複利效應
```
