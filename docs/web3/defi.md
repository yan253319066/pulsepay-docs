# DeFi 深入理解

## 深入 DeFi 机制

### AMM（自动做市商）原理

```
传统交易所：
- 订单簿匹配
- 买家和卖家撮合

AMM（Uniswap/PancakeSwap）：
- 无需订单簿
- 算法自动定价
- 流动性池保证交易
```

###流动性池 (Liquidity Pool)

```
池子结构：
┌─────────────────────────────────┐
│     USDC/USDT 流动性池           │
├─────────────────────────────────┤
│  500,000 USDC + 500,000 USDT    │
│  总价值：$1,000,000              │
│  比例：50:50                    │
└─────────────────────────────────┘

交易者：
- 想要 USDT → 放入 USDC → 获得 USDT
- 汇率由算法自动计算
```

### 无常损失 (Impermanent Loss)

```
什么是无常损失？

你存入流动性池：
- 10 ETH（存入时 $2000）
- 收到 LP Token

如果 ETH 涨到 $4000：
- 你池子里的 ETH 只剩 7 个
- 你还有 14,000 USDT

对比持有不动：
- 持有 10 ETH = $40,000

无常损失：$40,000 - $40,000 = 0？（实际计算更复杂）
```

## DeFi 收益来源

### 1. 交易手续费

```
AMM 每笔交易收取 0.1-0.3%：

示例：
- PancakeSwap: 0.25% 手续费
- 0.17% 给 LP
- 0.08% 给 CAKE 质押者
```

### 2. 借贷利息

```
Aave/Compound：

借款人支付利息
出借人获得利息
平台抽成

利率由供需决定：
- 借款需求高 → 利率上升
- 存款多 → 利率下降
```

### 3. 质押奖励

```
PoS 区块链的验证者奖励：

ETH 2.0 验证者：
- 年化收益约 4-5%
- 来源：新区块奖励

BSC 验证者：
- 年化收益约 5-10%
- 来源：交易手续费 + 区块奖励
```

### 4. 流动性激励

```
协议为了吸引流动性：

发行自己的 Token 奖励 LP

示例：
PancakeSwap：
- 交易手续费 0.25%
- 额外 CAKE 奖励

Yearn Finance：
- 策略收益
- YFI 治理代币奖励
```

## DeFi 风险详解

### 1. 智能合约风险

```
风险：
代码漏洞可能导致资金被盗

历史案例：
- The DAO 攻击：$60M 被盗
- Poly Network：$611M 被盗（后归还）

防范：
- 选择知名协议
- 查看代码审计
- 不要 all in 一个协议
```

### 2. 无常损失（深入）

```
数学计算：

存入时：
- ETH = $2000
- 你存入 1 ETH + 2000 USDC

后来：
- ETH = $4000（2x）
- 池子变成：0.707 ETH + 2828 USDC
- 总价值 = $2828 + $2828 = $5656

持有不动：
- 1 ETH = $4000
- + 2000 USDC = $6000

无常损失 = $6000 - $5656 = $344 (5.7%)
```

### 3. 预言机风险

```
预言机 = 提供外部价格数据

风险：
- 预言机被操纵
- 价格数据延迟
- 导致套利/清算攻击

案例：
- Mango Markets：$117M 被攻击
- 利用预言机操纵价格

防范：
- 使用多个预言机
- 时间加权平均价格（TWAP）
```

### 4. 流动性风险

```
风险：
- 协议 TVL 下降
- 大额交易滑点高
- 难以退出

防范：
- 选择 TVL 高的协议
- 注意流动性集中度
```

## DeFi 策略

### 1. 稳定币理财

```
目标：稳定收益，无本金损失风险

策略：
1. Aave 存款：~3-5% APY
2. Curve 稳定币池：~3-8% APY
3. 流动性池做市：~5-15% APY

风险：智能合约风险、预言机风险
```

### 2. 流动性挖矿

```
目标：高收益

策略：
1. 在 DEX 提供流动性
2. 获得 Token 奖励
3. 及时卖出奖励

风险：无常损失 + Token 贬值

技巧：
- 选择 TVL 高的池子
- 关注 Token 解锁日程
- 设置止盈止损
```

### 3. 杠杆收益

```
目标：放大收益

策略：
1. 存入抵押品
2. 借出资产
3. 将借出资产再存入
4. 循环操作

风险：清算风险、利差风险

示例：
- 存入 1000 USDC
- 借出 700 USDT
- 再存入 USDT 获取收益
- 实际敞口：1700 USDT 等效
```
