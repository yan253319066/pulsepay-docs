# DeFiとは？

## 一言で理解する DeFi

**DeFi = Decentralized Finance = 分散型金融**

従来の銀行機能をコードでブロックチェーンに移設：
- 預金 → 利息を獲得
- 借入 → 利息を支払う
- 取引 → 資産を交換
- 投資 → 収益を獲得

```
従来の銀行：
あなた → 銀行に預金 → 銀行が融資 → 銀行が差价を赚得 → あなたに小额の利息

DeFi：
あなた → スマートコントラクトに存款 → スマートコントラクトが自動融资 → アルゴリズムが金利決定 → 収益はあなたに
```

## DeFi vs 従来の金融

| 項目 | 従来の金融 | DeFi |
|------|------------|------|
| **参入門槛** | 本人確認、银行口座が必要 | ウォレットアドレスだけでOK |
| **サービス時間** | 平日9:00-17:00 | 24時間365日 |
| **地理的制限** | 跨境が困难 | 世界中で利用可能 |
| **透明性** | 黑箱操作 | コード开源、チェーン上で検証可能 |
| **收益率** | 年率1-5% | 年率5-100%以上 |
| **管理権** | 第三者管理 | 自分の資産自己管理 |
