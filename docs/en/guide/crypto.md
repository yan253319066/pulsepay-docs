# Crypto Basics

## What is Cryptocurrency?

Cryptocurrency is a digital asset based on blockchain technology, not controlled by any government or bank.

```
Traditional Currency:
USD/CNY → Government credit → Bank accounting → You hold

Cryptocurrency:
BTC/ETH/USDT → Code is law → Blockchain accounting → You hold (via private key)
```

## Major Cryptocurrencies

### 🟠 Bitcoin (BTC) - Digital Gold

```
Founder: Satoshi Nakamoto (anonymous)
Launch: 2009
Total Supply: 21 million (fixed)
Position: Digital gold, value storage
```

### 🔷 Ethereum (ETH) - Digital Oil

```
Founder: Vitalik Buterin
Launch: 2015
Total Supply: No cap (has burn mechanism)
Position: Smart contract platform
```

### 📝 Stablecoins (USDT/USDC)

```
Feature: Pegged to USD
1 USDT ≈ 1 USD
Purpose: Value storage, payment medium
```

## Wallet Basics

### 🔑 Private Key vs Public Key vs Address

```
Private Key
   ↓ Encryption
Public Key
   ↓ Hash
Address = Your bank account
```

**Important**:
- Private Key = Bank password (keep secret!)
- Public Key = Bank account number (can share)
- Address = Receiving account (can share)

## Security Tips

```
✅ Security checklist:

1. Never tell anyone your private key/seed phrase
2. Don't click suspicious links
3. Verify address before transferring
4. Use hardware wallet for large amounts
5. Enable two-factor authentication
6. Don't chase high returns (Ponzi warning)
```
