# 加密貨幣入門

## 什麼是加密貨幣？

### 一句話解釋

```
加密貨幣 = 區塊鏈上的數位資產

類比：
- 比特幣 = 數位黃金（價值儲存）
- 以太坊 = 數位石油（燃料+應用）
- USDT = 數位美元（穩定幣）
```

## 加密貨幣 vs 傳統貨幣

| 維度 | 傳統貨幣 | 加密貨幣 |
|------|----------|----------|
| **發行方** | 各國央行 | 區塊鏈協議 |
| **供應量** | 可無限印鈔 | 固定或算法控制 |
| **轉帳** | 需要銀行/中介 | 點對點，無中介 |
| **隱私** | 實名制 | 匿名/假名 |
| **可用性** | 受時間地點限制 | 24/7，全球通用 |

## 主要加密貨幣

### 比特幣 (BTC)

```
定位：數位黃金
發行量：2100萬枚（固定）
用途：價值儲存、抗通脹
共識機制：工作量證明 (PoW)
```

### 以太坊 (ETH)

```
定位：數位石油+智能合約平臺
用途：支付 Gas 費、運行 DApp
特點：支援智能合約
共識機制：利害關係證明 (PoS)
```

### 穩定幣 (USDT/USDC)

```
定位：區塊鏈上的美元
特點：1:1 錨定美元
用途：交易媒介、價值穩定
類型：
- USDT (Tether)
- USDC (Circle)
```

### BNB (幣安幣)

```
定位：幣安生態代幣
用途：支付 BSC 鏈 Gas 費
特點：折扣手續費
用途：幣安交易所手續費減免
```

## 如何獲得加密貨幣

### 方法一：交易所購買

```
1. 選擇交易所
   - 幣安 (Binance)
   - OKX
   - Gate.io

2. 完成 KYC 認證
   - 身份驗證
   - 銀行卡/信用卡綁定

3. 購買加密貨幣
   - 支持銀行卡直接購買
   - 支持 USDT 購買
```

### 方法二：P2P 交易

```
P2P = 點對點交易
用戶直接與另一方交易
平臺作為托管保障

優點：
- 無 KYC 要求
- 可議價
- 本地交易
```

### 方法三：挖礦/質押

```
挖礦 (PoW)：
- 消耗算力獲得獎勵
- 比特幣、以太坊經典

質押 (PoS)：
- 質押代幣獲得獎勵
- 乙太坊 2.0、Cardano、Solana
```

## 如何存放加密貨幣

### 交易所錢包

```
適合：新手、日常交易
優點：方便、易用
缺點：資產由交易所保管

警告：
- 交易所可能倒閉
- 可能被盜
- 不是你的鑰匙
```

### 自托管錢包

```
適合：長期持有、安全要求高
優點：完全控制資產
缺點：需要自己保管私鑰

類型：
- 熱錢包（手機/網頁）
- 冷錢包（硬體）
```

## 加密貨幣的風險

### 市場風險

```
波動性：
- 比特幣一天可能漲跌 10-20%
- 山寨幣可能漲跌 50%+
- 可能一夜暴富，也可能一夜歸零

流動性風險：
- 小幣種可能無法變現
- 高峰期網絡擁堵
```

### 安全風險

```
盜竊風險：
- 交易所被駭
- 錢包被盜
- 私鑰洩露

騙局風險：
- 土狗代幣（毫無價值）
- 龐氏騙局
- 釣魚網站
```

### 監管風險

```
各國政策不同：
- 中國：禁止交易挖礦
- 美國：加強監管
- 日本：合法化
- 監管變化可能影響價格
```

## 安全建議

### ✅ 建議做的事

```
1. 使用強密碼+2FA
   - Google Authenticator 優於短信

2. 分散存儲
   - 大額資產放冷錢包
   - 小額放熱錢包

3. 驗證網址
   - 確認是官網
   - 避免釣魚網站

4. 小額嘗試
   - 先用小額測試
   - 確認流程正確
```

### ❌ 千萬不要做的事

```
1. 透露私鑰/助記詞
   - 任何人都不能告訴
   - 官方永遠不會問你要

2. 點擊陌生連結
   - 可能導向釣魚網站
   - 使用書籤訪問交易所

3. 盲目跟風
   - DYOR (Do Your Own Research)
   - 不要 FOMO 入場
```
