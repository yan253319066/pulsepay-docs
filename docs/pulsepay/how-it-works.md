# PulsePay 工作原理

## 整体架构

```
┌─────────────────────────────────────────────────────────────┐
│                        用户层                                │
│                   PulseWallet (app.pulsepay.fun)           │
│              用户管理资产、参与收益分成                        │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                    AI Gateway 层                            │
│                   ai.pulsepay.fun                          │
│            统一 AI 调用入口，支持多种模型                      │
│            • OpenAI GPT-4 • Claude 3 • Gemini              │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                      支付层                                 │
│                    pulsepay.fun                             │
│              USDT/BNB 充值、消费、提现                       │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                      协议层                                 │
│              AIP Token + 收益分成合约                        │
│         收入自动分配，链上透明可查                           │
└─────────────────────────────────────────────────────────────┘
```

## 用户使用流程

### 第一步：注册与充值

```
1. 访问 app.pulsepay.fun
2. 创建或导入钱包
3. 充值 USDT/BNB

支持的链：
- BSC (BEP-20)
- TRON (TRC-20)
- ETH (ERC-20)
```

### 第二步：获取 API Key

```
1. 访问 ai.pulsepay.fun
2. 注册/登录账号
3. 充值余额
4. 获取 API Key

API Key 格式：
ppa_live_xxxxxxxxxxxxxxxxxxxx
```

### 第三步：调用 AI

```
开发者使用 API：

curl https://api.pulsepay.fun/v1/ai/chat \
  -H "X-API-TOKEN: ppa_live_xxxx" \
  -H "Content-Type: application/json" \
  -d '{"model": "gpt-4", "messages": [...]}'

按量计费，自动扣费
```

### 第四步：获得收益（持有 AIP）

```
持有 AIP → 质押参与收益分成
        ↓
每日获得平台收入的分红 (USDT/BNB)
        ↓
持续获得，被动收入
```

## 收益分配机制

### 收入来源

```
AI Gateway 收入 = AI 调用手续费
支付系统收入 = 支付处理手续费

每笔交易：
- 用户支付 100 USDT
- 平台收取 1 USDT 手续费 (1%)
- 这 1 USDT 就是"收入"
```

### 分配比例

```
每日的平台收入分配：

┌────────────────────────────────────────┐
│            100% 平台收入               │
├────────────────────────────────────────┤
│                                        │
│   40% ─────────────────→ AIP 质押者分红 │
│                                        │
│   30% ─────────────────→ 回购销毁 AIP  │
│                                        │
│   20% ─────────────────→ 团队运营      │
│                                        │
│   10% ─────────────────→ 生态激励     │
│                                        │
└────────────────────────────────────────┘
```

### 计算示例

```
假设：
- 今日平台收入：10,000 USDT
- 收益分成比例：40%
- 你的 AIP 质押量：10,000 AIP
- 全网 AIP 质押总量：1,000,000 AIP

计算：
┌─────────────────────────────────────────────┐
│ 可分配收益：10,000 × 40% = 4,000 USDT      │
├─────────────────────────────────────────────┤
│ 你的份额：10,000 / 1,000,000 = 1%           │
├─────────────────────────────────────────────┤
│ 你的收益：4,000 × 1% = 40 USDT              │
└─────────────────────────────────────────────┘

年化收益（假设每日收入不变）：
- 每日：40 USDT
- 每年：14,600 USDT
- 年化：146%（基于 10,000 USDT 投资）
```

## AIP Token 的价值支撑

### 1. 使用价值

```
持有 AIP = 使用平台的"门票"

- AIP 质押参与收益分成
- 未来可能用于支付 AI 服务
- 持有越多，收益分成比例可能越高
```

### 2. 通缩机制

```
30% 收入用于回购销毁

机制：
1. 每日用 30% 收入从市场回购 AIP
2. 销毁这些 AIP（永久移除）
3. 流通量减少
4. 供给减少 → 价值支撑
```

### 3. 需求驱动

```
用户需要 AIP 才能：
- 参与收益分成
- 分享平台增长红利

需求来源：
- AI 用户 → 想参与分红 → 买 AIP
- 投资者 → 看好项目 → 买 AIP
- 生态激励 → 吸引更多用户
```

## 智能合约架构

### AIP Token 合约

```
标准：ERC-20 (BSC 链)

主要功能：
- 转账：transfer()
- 授权：approve()
- 快照：记录质押时的代币余额

合约地址：可在 BSCScan 查看
```

### 收益分成合约

```
主要功能：
- stake()：质押 AIP
- unstake()：解除质押
- claim()：领取收益
- getReward()：计算应得收益

特点：
- 无锁定期
- 随时可退出
- 收益实时计算
```

### 收入管理合约

```
主要功能：
- 收集每日收入
- 自动分配（40%/30%/20%/10%）
- 链上记录每笔分配

透明性：
- 任何人都可以在链上验证
- 合约代码开源
```

## 参与方式

### Step 1：准备工作

```
需要的工具：
1. 加密钱包（MetaMask 或 PulseWallet）
2. 少量 BNB（支付 Gas 费）
3. USDT（用于兑换 AIP）
```

### Step 2：获取 AIP

```
方式一：官方兑换
1. 访问 stake.pulsepay.fun
2. 连接钱包
3. 使用 USDT/BNB 兑换 AIP
4. Phase 1 固定价格：1 USDT = 1 AIP

方式二：二级市场
- Phase 2 开放后
- 可在 DEX 交易
```

### Step 3：质押参与

```
1. 访问 stake.pulsepay.fun
2. 连接钱包
3. 质押 AIP
4. 最低 10 AIP 起
5. 开始获得每日收益
```

### Step 4：领取收益

```
1. 查看收益仪表盘
2. 点击"领取"按钮
3. 收益转入钱包

收益发放：
- 自动计算每日收益
- 需要手动点击领取
- 可以随时取消质押
```
