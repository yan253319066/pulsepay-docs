## Get Help

### Report Issues

```
Having problems?

1. Check FAQ first
   The FAQ section in this documentation

2. Search community
   Search keywords in Discord/Telegram

3. Ask questions
   Post in appropriate community channels
   - #support channel
   - #questions channel

Please provide:
- Problem description
- Screenshots
- Transaction TxID (if applicable)
```

### Bug Reporting

```
Found a bug or security issue?

1. Regular bugs
   - Discord #bugs channel
   - Detailed reproduction steps

2. Security issues
   - Send to security@pulsepay.fun
   - Keep details private

3. Smart contract vulnerabilities
   - Bug Bounty program
   - See security page for details
```
