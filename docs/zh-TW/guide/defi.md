# DeFi 是什麼？

## 一句話理解 DeFi

**DeFi = Decentralized Finance = 去中心化金融**

將傳統銀行功能移到區塊鏈上，用代碼實現：
- 存錢 → 賺利息
- 借錢 → 支付利息
- 交易 → 交換資產
- 投資 → 賺取回報

```
傳統銀行：
你 → 存款到銀行 → 銀行借給別人 → 銀行賺差價 → 你拿微薄利息

DeFi：
你 → 存款到智能合約 → 智能合約自動借出 → 演算法決定利率 → 回報給你
```

## DeFi vs 傳統金融

| 維度 | 傳統 | DeFi |
|------|------|------|
| **準入門檻** | 需要身份驗證、银行帳戶 | 只需要錢包地址 |
| **服務時間** | 週一至週五 9:00-17:00 | 24/7 |
| **地理限制** | 跨境困難 | 全球通用 |
| **透明度** | 黑盒子 | 開源，鏈上可驗證 |
| **收益** | 1-5% 年化 | 5-100%+ 年化 |
| **控制權** | 第三方保管 | 自己保管 |

## 核心 DeFi 應用

### 1. 💰 借貸/借款

**賺取利息：**
```
傳統銀行：
存款利率 ≈ 0.3%/年

DeFi (Aave/Compound)：
穩定幣存款利率 ≈ 3-8%/年
```

**借款：**
```
傳統銀行：
需要信用審查
審批時間長
利率固定

DeFi：
超額抵押即可借款
秒級審批
利率市場化
```

### 2. 🔄 交易 (DEX)

**中心化交易所 vs DEX：**
```
中心化交易所 (Binance)：
- 你把錢存到交易所
- 交易所記錄
- 需要 KYC
- 可能被凍結

DEX (Uniswap/PancakeSwap)：
- 直接用錢包交易
- 智能合約自動撮合
- 匿名，無需註冊
- 無法被凍結
```

### 3. 📈 收益農業 (Yield Farming)

```
簡單理解：把錢放到 DeFi 協議
        ↓
        自動尋找最高收益策略
        ↓
        複利效應

風險等級：
- 低風險：穩定幣借貸 (3-10%)
- 中風險：流動性提供 (10-50%)
- 高風險：收益農場 (50%+) 
```

### 4. 🏦 流動性挖礦

```
流動性 = 交易對的深度

為 DEX 提供流動性：
- 你存入兩種代幣
- 交易者支付手續費
- 你獲得手續費分成

風險：
- 無常損失 (Impermanent Loss)
- 代幣貶值
```

## DeFi 安全須知

### ⚠️ 智能合約風險

```
DeFi 基於智能合約
智能合約可能有漏洞

歷史事故：
- The DAO 攻擊 (2016)
- 多個 DeFi 協議被盜

建議：
- 選擇知名協議
- 查看審計報告
- 不要投入全部資金
```

### ⚠️ 流動性風險

```
無常損失 (Impermanent Loss)：
存入流動性池後
如果代幣價格變動
你實際收到的價值可能減少

預防：
- 只提供穩定幣對流動性
- 了解風險後再操作
```

### ⚠️ Rug Pull 風險

```
Rug Pull = 開發者捲款跑路

常見模式：
1. 建立 DeFi 專案
2. 吸引流動性
3. 拋售代幣
4. 投資者損失全部

預防：
- 查看團隊背景
- 合約是否已審計
- 是否有鎖倉機制
```
