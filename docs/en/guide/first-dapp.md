# First DApp Tutorial

## What is a DApp?

```
DApp = Decentralized Application

Analogy:
App = App on phone (WeChat, Douyin)
     ↓
App data stored on company server

DApp = App in browser
     ↓
DApp data stored on blockchain
```

## Prerequisites

### Tools needed

```
1. Browser wallet (MetaMask recommended)
   👉 metamask.io

2. Some crypto (for Gas fees)
   Recommended: BNB (for BSC chain)

3. Stablecoin (USDT) for trading
   Suggestion: Start with small amount
```

### Step 1: Install MetaMask

```
1. Visit metamask.io
2. Click "Download"
3. Select your browser
4. Install extension
5. Click "Get Started"
6. Choose "Create New Wallet"
7. Set password
8. Save seed phrase (12 words)
```

### Step 2: Switch to BSC Network

MetaMask defaults to Ethereum network. Switch to BSC:

```
1. Open MetaMask
2. Click network selector at top
3. Click "Add Network"
4. Fill BSC mainnet info:

   Network Name: BSC
   RPC URL: https://bsc-dataseed.binance.org/
   Chain ID: 56
   Symbol: BNB
   Block Explorer: https://bscscan.com
```

## Hands-on: Use PulsePay

### Step 1: Connect stake.pulsepay.fun

1. Open browser
2. Visit [stake.pulsepay.fun](https://stake.pulsepay.fun)
3. Click "Connect Wallet"
4. Select MetaMask
5. Sign to confirm connection

### Step 2: Deposit USDT

Withdraw USDT from exchange (e.g., Binance) to your MetaMask wallet:

1. Copy your MetaMask wallet address
2. On exchange, select USDT withdrawal
3. Choose BSC (BEP-20) network
4. Paste your wallet address
5. Confirm withdrawal

### Step 3: Swap for AIP

1. On [stake.pulsepay.fun](https://stake.pulsepay.fun), click "Swap"
2. Select USDT to AIP swap
3. Enter amount (minimum 10 USDT)
4. Confirm swap
5. Wait for blockchain confirmation

### Step 4: Participate in Revenue Share

1. On [stake.pulsepay.fun](https://stake.pulsepay.fun), click "Revenue Share"
2. Activate AIP Revenue Share
3. Minimum 10 AIP
4. Confirm transaction
5. Start earning daily returns

## Security Checklist

```
✅ Before each transaction:
□ Correct target address
□ Correct network (BSC)
□ Correct amount
□ Reasonable Gas fee

✅ Precautions:
□ Don't click unknown links
□ Verify contract address
□ Test with small amount first
□ Don't chase high return promises
```
