## 通縮機制

### 回購銷毀

```
30% 的平台收入用於回購銷毀

機制：
1. 每日用 USDT 從市場回購 AIP
2. 將回購的 AIP 發送到銷毀地址
3. 銷毀的 AIP 永久移除

效果：
- 流通量減少
- 供給減少
- 價值支撐
```

### 銷毀地址

```
黑洞地址（Burn Address）：
0x000000000000000000000000000000000000dEaD

特點：
- 沒有人有私鑰
- 發送到這裡的代幣永遠無法取出
- 等同於永久銷毀
```

### 通縮計算示例

```
假設：
- 每日平台收入：10,000 USDT
- 回購銷毀：3,000 USDT
- AIP 價格：1 USDT

每日銷毀量：3,000 AIP

年銷毀量：3,000 × 365 = 1,095,000 AIP
年銷毀比例：1,095,000 / 65,000,000 = 1.68%

長期看：
- 流通量持續減少
- 稀缺性增加
- 價值支撐
```

## 常見問題

### Q：為什麼初始銷毀 10%？

A：初始銷毀是為了：
- 減少流通量
- 價值支撐
- 表明團隊信心

### Q：AIP 會通貨膨脹嗎？

A：不會。AIP 是固定總量，沒有增發機制。實際會發生通縮（回購銷毀）。

### Q：Phase 1 沒來得及怎麼辦？

A：Phase 2 開放後可以在 DEX 購買，但價格將由市場決定，可能高於或低於 1 USDT。

### Q：團隊代幣解鎖會砸盤嗎？

A：不會，因為：
- 有 1 年鎖倉
- 之後線性釋放 2 年
- 團隊與項目長期綁定
