# 快速开始指南

## 准备工作

### 需要准备

```
1. 加密钱包
   推荐：MetaMask
   安装：浏览器插件或手机 App

2. 少量加密货币
   BNB：约 0.01 BNB（支付 Gas 费）
   USDT：用于兑换 AIP（最低 10 USDT）

3. 一个交易所账号
   用于购买 USDT/BNB
   推荐：币安、OKX、Gate
```

### 第一步：获取钱包

**方案一：MetaMask（浏览器插件）**

1. 访问 [MetaMask 官网](https://metamask.io)
2. 下载浏览器插件
3. 创建钱包
4. 备份助记词（12个单词）
5. 设置密码

**方案二：MetaMask（推荐）**

MetaMask 是最流行的 Web3 钱包，可以完全支持 PulsePay 的所有功能。

### 第二步：配置 BSC 网络

MetaMask 添加 BSC 网络：

1. 点击 MetaMask 插件
2. 点击网络选择器
3. 点击 "添加网络"
4. 填写以下信息：

   - 网络名称：BSC
   - RPC URL：`https://bsc-dataseed.binance.org/`
   - 链 ID：56
   - 符号：BNB
   - 区块浏览器：[https://bscscan.com](https://bscscan.com)

5. 点击保存

### 第三步：获取 USDT

**方式一：交易所购买**

1. 登录币安/OKX
2. 进入 C2C 交易
3. 买入 USDT（选择 BNB 网络）
4. 提现到你的钱包地址

::: warning ⚠️ 注意
- 网络选择 BSC (BEP-20)
- 提现地址填写你的钱包地址
- 小额测试确认无误后再大额
:::

**方式二：别人转账**

从其他钱包转账 USDT (BSC) 到你的地址

## 参与 Phase 1

### 兑换 AIP

1. 访问 [stake.pulsepay.fun](https://stake.pulsepay.fun)
2. 点击 "连接钱包"
3. 选择 MetaMask
4. 授权连接
5. 进入 "兑换" 页面
6. 输入 USDT 数量
   - 最低 10 USDT
   - 最高 10,000 USDT/人
7. 确认兑换
   - 1 USDT = 1 AIP
   - 无额外手续费
8. 等待链上确认
9. 完成！AIP 已到账

### 激活 AIP 收益分成

1. 进入 "质押" 页面
2. 输入要质押的 AIP 数量
3. 点击 "质押"
4. 确认交易
5. 等待链上确认

::: tip 最低质押：10 AIP
完成！从明天开始获得每日收益
:::

## 领取收益

### 查看收益

1. 进入 "收益" 页面
2. 查看以下数据：
   - 累计收益
   - 今日收益
   - 待领取收益

**收益计算**：
- 每日 UTC 0 点结算
- 按质押比例分配

### 领取收益

1. 点击 "领取" 按钮
2. 确认领取数量
3. 点击确认
4. 等待链上确认
5. USDT/BNB 到账钱包

::: warning ⚠️ 注意
- 收益需要手动领取
- 不会自动到账
- 建议每周或每天领取一次
:::

## 常见问题

### Q：最低需要多少钱开始？

A：
- 最低兑换：10 USDT
- Gas 费：约 0.005 BNB（约 $1-2）
- 建议准备：至少 15 USDT + 0.01 BNB

### Q：收益多久到账？

A：
- 每日 UTC 0 点结算
- 需要手动点击"领取"
- 领取后立即到账

### Q：可以随时退出吗？

A：
- 可以随时解除质押
- 解除后不再获得新收益
- 已领取的收益不受影响

### Q：忘记钱包密码怎么办？

A：
- 使用助记词恢复钱包
- 密码忘了可以重置
- 助记词忘了 = 资产永远丢失
