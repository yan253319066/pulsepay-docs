# 加密货币入门

## 什么是加密货币？

加密货币是基于区块链技术的数字资产，不受任何政府或银行控制。

```
传统货币：
美元/人民币 → 政府信用背书 → 银行记账 → 你持有

加密货币：
BTC/ETH/USDT → 代码即法律 → 区块链记账 → 你持有（通过私钥）
```

## 主流加密货币

### 🟠 比特币 (BTC) - 数字黄金

```
创始人：中本聪（匿名）
诞生时间：2009年
总量：2100万枚（有限供应）
定位：数字黄金，价值存储
```

**类比**：比特币就像区块链世界的黄金，数量有限，价值稳定。

### 🔷 以太坊 (ETH) - 数字石油

```
创始人：Vitalik Buterin
诞生时间：2015年
总量：无上限（但有销毁机制）
定位：智能合约平台
```

**类比**：以太坊就像区块链世界的石油，是运行程序的"燃料"。

### 📝 稳定币 (USDT/USDC) - 数字美元

```
特点：价值锚定美元
1 USDT ≈ 1 美元
用途：价值储存、交易媒介
```

**类比**：稳定币就像区块链世界的美元，稳定但不会"升值"。

## 钱包基础

### 🔑 私钥 vs 公钥 vs 地址

```
私钥（Secret Key）
   ↓ 加密运算
公钥（Public Key）
   ↓ 哈希运算
地址（Address）= 你的银行账号
```

**重要**：
- 私钥 = 银行卡密码（绝对保密！）
- 公钥 = 银行卡号（可以给别人）
- 地址 = 收款账号（可以给别人）

### 钱包类型

| 类型 | 例子 | 特点 |
|------|------|------|
| 热钱包 | MetaMask、手机钱包 | 方便但风险高 |
| 冷钱包 | 硬件钱包（Ledger） | 安全但不便 |
| 交易所钱包 | 币安钱包 | 方便但需信任平台 |

### 助记词 (Seed Phrase)

```
12个单词的备份
示例：
apple banana cherry dog elephant frog 
grape horse ice juice kiwi lemon

⚠️ 重要性：
- 丢失 = 永远失去资产
- 泄露 = 资产被盗
- 必须手写保存多个副本
```

## 交易基础

### Gas 费是什么？

```
Gas = 区块链的"手续费"

类比：
- 比特币/以太坊 = 高速公路收费
- Gas = 你付的过路费
- BSC = 低收费高速公路（PulsePay用的就是这个）
```

### 交易确认

```
发起交易 → 等待打包 → 链上确认 → 完成

确认数越多越安全：
- 1个确认：基本安全（小额）
- 6个确认：标准安全（一般转账）
- 12个确认：高度安全（大额）
```

## 风险提示

### ⚠️ 市场风险
- 加密货币价格波动剧烈
- 可能一天涨跌 10%-30%
- 只投资你能承受损失的钱

### ⚠️ 安全风险
- 私钥丢失 = 资产永远没了
- 钓鱼网站 / 诈骗
- 智能合约漏洞

### ⚠️ 监管风险
- 各国政策不同
- 可能影响使用
