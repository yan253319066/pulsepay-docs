# Get Started with PulsePay

## Prerequisites

### What You Need

```
1. Crypto Wallet
   Recommended: MetaMask or PulseWallet
   Install: Browser extension or mobile app

2. Small Amount of Crypto
   BNB: ~0.01 BNB (for Gas fees)
   USDT: For exchanging AIP (minimum 10 USDT)

3. Exchange Account
   For buying USDT/BNB
   Recommended: Binance, OKX
```

### Step 1: Get a Wallet

```
Option 1: MetaMask (Browser Extension)

1. Visit metamask.io
2. Download browser extension
3. Create wallet
4. Backup seed phrase (12 words)
5. Set password

Option 2: PulseWallet (Recommended)

1. Visit app.pulsepay.fun
2. Click "Create Wallet"
3. Set password
4. Backup seed phrase
5. Done
```

### Step 2: Configure BSC Network

```
Add BSC Network in MetaMask:

1. Click MetaMask extension
2. Click network selector
3. Click "Add Network"
4. Fill in:

Network Name: BSC
RPC URL: https://bsc-dataseed.binance.org/
Chain ID: 56
Symbol: BNB
Block Explorer: https://bscscan.com

5. Click Save
```

### Step 3: Get USDT

```
Method 1: Buy from Exchange

1. Login to Binance/OKX
2. Go to C2C Trading
3. Buy USDT (select BNB network)
4. Withdraw to your wallet address

⚠️ Note:
- Select BSC (BEP-20) network
- Fill in your wallet address
- Test with small amount first
```

## Participate in Phase 1

### Exchange AIP

```
1. Visit stake.pulsepay.fun
2. Click "Connect Wallet"
3. Select MetaMask or PulseWallet
4. Authorize connection

5. Go to "Exchange" page
6. Enter USDT amount
   - Minimum 10 USDT
   - Maximum 10,000 USDT/person

7. Confirm exchange
   - 1 USDT = 1 AIP
   - No extra fees

8. Wait for on-chain confirmation
9. Done! AIP is in your wallet
```

### Stake AIP

```
1. Go to "Stake" page
2. Enter AIP amount to stake
3. Click "Stake"
4. Confirm transaction
5. Wait for on-chain confirmation

⚠️ Minimum: 10 AIP

Done! Start earning daily returns from tomorrow
```

## Claim Returns

### View Returns

```
1. Go to "Returns" page
2. View:
   - Cumulative returns
   - Today's returns
   - Pending claims

Calculation:
- Daily settlement at UTC 0:00
- Distributed by stake ratio
```

### Claim Returns

```
1. Click "Claim" button
2. Confirm claim amount
3. Click confirm
4. Wait for on-chain confirmation
5. USDT/BNB to wallet

⚠️ Note:
- Returns need manual claiming
- Not automatic
- Recommend claiming weekly or daily
```
