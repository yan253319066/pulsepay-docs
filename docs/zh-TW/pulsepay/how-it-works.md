# PulsePay 工作原理

## 整體架構

```
┌─────────────────────────────────────────────────────────────┐
│                        用戶層                                │
│                   MetaMask / 第三方錢包                        │
│              用戶管理資產、參與收益分成                        │
└─────────────────────────────────────────────────────────────┘
                               ↓
┌─────────────────────────────────────────────────────────────┐
│                    AI Gateway 層                            │
│                   ai.pulsepay.fun                          │
│            統一 AI 調用入口，支持多種模型                      │
│            • OpenAI GPT-4 • Claude 3 • Gemini              │
└─────────────────────────────────────────────────────────────┘
                               ↓
┌─────────────────────────────────────────────────────────────┐
│                      支付層                                 │
│                    pulsepay.fun                             │
│              USDT/BNB 充值、消費、提現                       │
└─────────────────────────────────────────────────────────────┘
                               ↓
┌─────────────────────────────────────────────────────────────┐
│                      協議層                                 │
│              AIP Token + 收益分成合約                        │
│         收入自動分配，鏈上透明可查                           │
└─────────────────────────────────────────────────────────────┘
```

## 用戶使用流程

### 第一步：註冊與充值

```
1. 訪問 app.pulsepay.fun
2. 創建或導入錢包
3. 充值 USDT/BNB

支持的鏈：
- BSC (BEP-20)
- TRON (TRC-20)
- ETH (ERC-20)
```

### 第二步：獲取 API Key

```
1. 訪問 ai.pulsepay.fun
2. 註冊/登錄帳號
3. 充值餘額
4. 獲取 API Key

API Key 格式：
ppa_live_xxxxxxxxxxxxxxxxxxxx
```

### 第三步：調用 AI

```
開發者使用 API：

curl https://api.pulsepay.fun/v1/ai/chat \
  -H "X-API-TOKEN: ppa_live_xxxx" \
  -H "Content-Type: application/json" \
  -d '{"model": "gpt-4", "messages": [...]}'

按量計費，自動扣費
```

### 第四步：獲得收益（啟動 AIP 收益分成）

```
持有 AIP → 質押參與收益分成
        ↓
每日獲得平台收入的分红 (USDT/BNB)
        ↓
持續獲得，被動收入
```

## 收益分配機制

### 收入來源

```
AI Gateway 收入 = AI 調用手續費
支付系統收入 = 支付處理手續費

每筆交易：
- 用戶支付 100 USDT
- 平台收取 1 USDT 手續費 (1%)
- 這 1 USDT 就是"收入"
```

### 分配比例

```
每日的平台收入分配：

┌────────────────────────────────────────┐
│            100% 平台收入               │
├────────────────────────────────────────┤
│                                        │
│   40% ─────────────────→ AIP 質押者分紅 │
│                                        │
│   30% ─────────────────→ 回購銷毀 AIP  │
│                                        │
│   20% ─────────────────→ 團隊運營      │
│                                        │
│   10% ─────────────────→ 生態激勵     │
│                                        │
└────────────────────────────────────────┘
```

### 計算示例

```
假設：
- 今日平台收入：10,000 USDT
- 收益分成比例：40%
- 你的 AIP 質押量：10,000 AIP
- 全網 AIP 質押總量：1,000,000 AIP

計算：
┌─────────────────────────────────────────────┐
│ 可分配收益：10,000 × 40% = 4,000 USDT      │
├─────────────────────────────────────────────┤
│ 你的份額：10,000 / 1,000,000 = 1%           │
├─────────────────────────────────────────────┤
│ 你的收益：4,000 × 1% = 40 USDT              │
└─────────────────────────────────────────────┘

年化收益（假設每日收入不變）：
- 每日：40 USDT
- 每年：14,600 USDT
- 年化：146%（基於 10,000 USDT 投資）
```

## AIP Token 的價值支撐

### 1. 使用價值

```
持有 AIP = 使用平台的"門票"

- AIP 質押參與收益分成
- 未來可能用於支付 AI 服務
- 持有越多，收益分成比例可能越高
```

### 2. 通縮機制

```
30% 收入用於回購銷毀

機制：
1. 每日用 30% 收入從市場回購 AIP
2. 銷毀這些 AIP（永久移除）
3. 流通量減少
4. 供給減少 → 價值支撐
```

### 3. 需求驅動

```
用戶需要 AIP 才能：
- 參與收益分成
- 分享平台增長紅利

需求來源：
- AI 用戶 → 想參與分紅 → 買 AIP
- 投資者 → 看好項目 → 買 AIP
- 生態激勵 → 吸引更多用戶
```

## 智能合約架構

### AIP Token 合約

```
標準：ERC-20 (BSC 鏈)

主要功能：
- 轉帳：transfer()
- 授權：approve()
- 快照：記錄質押時的代幣餘額

合約地址：可在 BSCScan 查看
```

### 收益分成合約

```
主要功能：
- stake()：啟動 AIP 收益分成
- unstake()：解除質押
- claim()：領取收益
- getReward()：計算應得收益

特點：
- 無鎖定期
- 隨時可退出
- 收益即時計算
```

### 收入管理合約

```
主要功能：
- 收集每日收入
- 自動分配（40%/30%/20%/10%）
- 鏈上記錄每筆分配

透明性：
- 任何人都可以在鏈上驗證
- 合約代碼開源
```

## 參與方式

### Step 1：準備工作

```
需要的工具：
1. 加密錢包（推薦 MetaMask）
2. 少量 BNB（支付 Gas 費）
3. USDT（用於兌換 AIP）
```

### Step 2：獲取 AIP

```
方式一：官方兌換
1. 訪問 stake.pulsepay.fun
2. 連接錢包
3. 使用 USDT/BNB 兌換 AIP
4. Phase 1 固定價格：1 USDT = 1 AIP

方式二：二級市場
- Phase 2 開放後
- 可在 DEX 交易
```

### Step 3：質押參與

```
1. 訪問 stake.pulsepay.fun
2. 連接錢包
3. 啟動 AIP 收益分成
4. 最低 10 AIP 起
5. 開始獲得每日收益
```

### Step 4：領取收益

```
1. 查看收益儀表盤
2. 點擊"領取"按鈕
3. 收益轉入錢包

收益發放：
- 自動計算每日收益
- 需要手動點擊領取
- 可以隨時取消質押
```
