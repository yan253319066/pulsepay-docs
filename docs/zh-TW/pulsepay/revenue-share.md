# 收益分成詳解

## 什麼是收益分成？

```
收益分成 = Revenue Share = 分享平台收入

持有 AIP → 質押參與 → 每日獲得 USDT/BNB 分紅

類比：
持有蘋果股票 → 獲得蘋果公司分紅
啟動 AIP 收益分成 → 獲得 PulsePay 平台分紅

區別：
- 股票分紅來自利潤
- AIP 分紅來自平台業務收入
```

## 如何參與？

### 參與條件

```
最低門檻：10 AIP

無鎖定期：隨時可質押/解除
無最高限制：想質押多少都可以

⚠️ 注意：
- 質押越多，分紅越多
- 解除質押需要等待
- 收益需要手動領取
```

### 質押步驟

```
1. 訪問 stake.pulsepay.fun
2. 連接錢包（MetaMask）
3. 進入"質押"頁面
4. 輸入 AIP 數量（或點擊最大值）
5. 點擊"質押"
6. 確認交易
7. 等待鏈上確認
8. 完成！開始獲得每日收益
```

### 領取收益

```
收益領取：

1. 進入"收益"頁面
2. 查看累計收益
3. 點擊"領取"
4. 收益轉入錢包

⚠️ 注意：
- 收益需要手動點擊領取
- 不會自動到帳
- 建議定期領取
```

### 解除質押

```
解除質押：

1. 進入"質押"頁面
2. 輸入解除數量
3. 點擊"解除質押"
4. 確認交易

⚠️ 注意：
- 解除後不再獲得新收益
- 已領取的收益不會收回
- 解除需要鏈上確認
```

## 收益計算

### 計算公式

```
個人每日收益 = (個人質押量 / 全網質押總量) × (當日平台收入 × 40%)
```

### 計算示例

```
假設條件：
- 今日平台收入：10,000 USDT
- 收益分成比例：40%
- 你的質押量：10,000 AIP
- 全網質押總量：1,000,000 AIP

計算過程：
1. 可分配收益 = 10,000 × 40% = 4,000 USDT
2. 你的份額 = 10,000 / 1,000,000 = 1%
3. 你的收益 = 4,000 × 1% = 40 USDT

結論：質押 10,000 AIP，每日獲得 40 USDT
```

### 年化收益計算

```
年化收益率計算：

日收益：40 USDT
年收益：40 × 365 = 14,600 USDT
投資本金：10,000 AIP（當時價值 10,000 USDT）
年化收益：14,600 / 10,000 = 146%

⚠️ 注意：
- 這是假設每日收入不變的情況
- 實際收入會波動
- 這是預期收益，不是保證收益
```

## 常見問題

### Q：收益是固定的嗎？

A：不是固定的。收益取決於：
- 每日平台收入
- 質押量佔全網比例
- 業務發展情況

### Q：需要每天領取嗎？

A：建議定期領取，但：
- 收益會累積，不會過期
- 可以一次性領取所有歷史收益
- 領取是免費的（無額外 Gas 費）

### Q：解除質押會影響已領取的收益嗎？

A：不影響。
- 已領取的收益歸你
- 解除質押只影響未來的新收益

### Q：質押有什麼風險？

A：主要風險：
- AIP 價格波動
- 平台業務下降

不存在：
- 不會爆倉
- 沒有清算
- 沒有懲罰

### Q：收益需要納稅嗎？

A：取決於你所在國家的法律。建議諮詢當地稅務顧問。
