# Web3進階

## 探索Web3世界

Web3是區塊鏈技術與互聯網的融合，帶來去中心化、透明和用戶主權的新時代。

## 核心概念

### DeFi（去中心化金融）

將傳統銀行功能移到區塊鏈上，用代碼實現：
- 存錢 → 賺利息
- 借錢 → 支付利息
- 交易 → 交換資產

詳情：[DeFi深入理解](./defi)

### 智能合約

自動執行的程序，運行在區塊鏈上：
- 無需中介
- 自動執行
- 不可篡改

詳情：[智能合約原理](./smart-contract)

### NFT

非同質化代幣，代表獨特的數字資產：
- 數字藝術
- 遊戲道具
- 會員資格

詳情：[NFT完全指南](./nft)

### DAO

去中心化自治組織：
- 社區治理
- 代幣投票
- 透明決策

詳情：[DAO組織](./dao)
